　$a$，$b$を実数とする．
3次方程式$x^3+ax^2+bx+1=0$は3つの複素数からなる解$\alpha_1$，$\alpha_2$，$\alpha_3$をもち，
相異なる$i$，$j$に対し$|\alpha_i-\alpha_j|=\sqrt3$をみたしている．
このような$a$，$b$の組をすべて求めよ．
