{\huge 2}
　$f(x)=x\sin x$ $(x \geqq 0)$とする．
点$\displaystyle \left( \frac{\pi}{2},\frac{\pi}{2} \right)$における$y=f(x)$の法線と，
$y=f(x)$のグラフの$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$の部分，
および$y$軸とで囲まれる図形を考える．
この図形を$x$軸の回りに回転して得られる回転体の体積を求めよ．
