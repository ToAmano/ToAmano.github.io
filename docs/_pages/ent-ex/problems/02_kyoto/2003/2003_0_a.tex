　三角形$ABC$と点$P$に対して，次の2つの条件は同値であることを証明せよ．
\begin{description}
\item[(i)]点$P$は三角形$ABC$の内部(周は除く)にある
\item[(ii)]正の数$a$，$b$，$c$があって，
$a\overrightarrow{PA}+b\overrightarrow{PB}+c\overrightarrow{PC}=\vec{0}$が成り立つ
\end{description}
\end{flushleft}
