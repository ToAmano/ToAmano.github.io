　7つの文字を並べた列$a_1a_2 \cdots a_7$で，次の3つの条件をみたすものの総数を求めよ．
\begin{description}
\item[(i)]$a_1,a_2,\cdots,a_7$は$A,B,C,D,E,F$のいずれかである
\item[(ii)]$i=1,2,\cdots,6$に対し，$a_i$と$a_{i+1}$は相異なる
\item[(iii)]$i=1,2,\cdots,6$に対し，$a_i$と$a_{i+1}$は右図において線分で結ばれている
\end{description}
\end{flushleft}
