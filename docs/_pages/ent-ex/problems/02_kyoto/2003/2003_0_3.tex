{\huge 3}
　四面体$OABC$は次の2つの条件
\begin{description}
\item[(i)]$\overrightarrow{OA}\bot\overrightarrow{BC}$，
$\overrightarrow{OB}\bot\overrightarrow{AC}$，
$\overrightarrow{OC}\bot\overrightarrow{AB}$
\item[(ii)]4つの面の面積がすべて等しい
\end{description}
をみたしている．
このとき，この四面体は正四面体であることを示せ．
\end{flushleft}
