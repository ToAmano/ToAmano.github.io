{\huge 4}
　複素数平面上で原点を$O$とし，1の表す点を$A$とする．
この平面上の点$P_1(z_1)$，$P_2(z_2)$，$P_3(z_3)$，$\cdots$が次の条件(i)，(ii)，(iii)を満たしているとき，
$|z_n|$の最大値を求めよ．
\begin{description}
\item[(i)]$\displaystyle z_1=\frac{i}{\sqrt3}$
\item[(ii)]$0^\circ<\arg z_{n+1}<\arg z_n<{360}^\circ$
\item[(iii)]線分$AP_n$の中点を$M_n$とするとき，
$n\geqq1$に対して$\triangle OP_nP_{n+1}\text{∽}\triangle OM_nA$が成立する．\\
ただしここでいう相似は，左辺の三角形と右辺の三角形の頂点が，書かれた順に対応しているものとする．
\end{description}
\end{flushleft}
