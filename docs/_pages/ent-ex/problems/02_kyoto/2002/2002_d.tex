{\huge 4}
　$f(x)$は$x^n$の係数が1である$x$の$n$次式である．
相異なる$n$個の有理数$q_1,q_2,\cdots,q_n$に対して$f(q_1),f(q_2),\cdots,f(q_n)$がすべて有理数であれば，
$f(x)$の係数はすべて有理数であることを，数学的帰納法を用いて示せ．
