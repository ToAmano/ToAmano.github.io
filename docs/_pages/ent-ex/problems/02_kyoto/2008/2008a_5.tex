　次の式で与えられる底面の半径が2，高さが1の円柱$C$を考える．
\[ C= \{ (x,y,z) | x^2+y^2 \leqq 4, \hspace{1zw} 0 \leqq x \leqq 1 \} \]
$xy$平面上の直線$y=1$を含み，$xy$平面と$45^\circ$の角をなす平面のうち，点$(0,2,1)$を通るものを$H$とする．
円柱$C$を平面$H$で二つに分けるとき，点$(0,2,0)$を含む方の体積を求めよ．
