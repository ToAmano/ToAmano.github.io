{\huge 3}
　$AB=AC$である二等辺三角形$ABC$を考える．
辺$AB$の中点を$M$とし，辺$AB$を延長した直線上に点$N$を，$AN:NB=2:1$となるようにとる．
このとき$\angle BCM = \angle BCN$となることを示せ．
ただし，点$N$は辺$AB$上にはないものとする．
\end{flushleft}
