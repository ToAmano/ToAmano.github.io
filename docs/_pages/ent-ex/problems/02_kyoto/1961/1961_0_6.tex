{\huge 6}
　直角二等辺三角形$ABC$の直角頂$A$をとおって$BC$に平行線をひき，その上に一点$D$をとり，$BD=BC$となるようにする．
\begin{description}
\item[(i)]$\angle ABD$は何度か．
\item[(ii)]$BD$，$AC$の交点を$E$とし，$BC$に関する$D$の対称点を$F$とすれば，$EF$は$DC$に平行であることを証明せよ．
\end{description}
