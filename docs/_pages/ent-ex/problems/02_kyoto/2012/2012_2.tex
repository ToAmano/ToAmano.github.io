　正四面体$OABC$において，点$P$，$Q$，$R$をそれぞれ
辺$OA$，$OB$，$OC$上にとる．
ただし$P$，$Q$，$R$は四面体$OABC$の頂点とは異なるとする．
$\triangle PRQ$が正三角形ならば，
3辺$PQ$，$QR$，$RP$はそれぞれ3辺$AB$，$BC$，$CA$に平行であることを証明せよ．
