{\huge 6}
　$9u_{n+1}=(10-{u_n}^2)u_n$ $(n=1,2,\cdots)$で定められる数列$\{u_n\}$がある．
いま，$0<u_1<3/2$と仮定するとき次のことを示せ．
\begin{description}
\item[(i)]すべての$n$について，$\displaystyle0<u_n<\frac{3}{2}$．
\item[(ii)]$u_1>u_2$ならばすべての$n$について$u_n>u_{n+1}$，$u_1<u_2$ならばすべての$n$について$u_n<u_{n+1}$．
\end{description}
\end{flushleft}
