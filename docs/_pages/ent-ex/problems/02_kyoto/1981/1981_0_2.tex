　$p$を有理数とし，次の関係をもつ$x_n$，$y_n$を座標にもつ平面上の点$P_n$ $(n=1,2,\cdots\cdots)$を考える；
\[ x_{n+1}=x_n+p(y_{n+1}+y_n), \hspace{1zw} y_{n+1}=y_n-p(x_{n+1}+x_n) \]
　いま，$x_1$，$y_1$がともに有理数で，かつ$P_1$は原点ではないとする．
このとき，すべての$x_n$，$y_n$は有理数であり，点$P_n$は原点を中心とする定円上にあることを示せ．
\end{flushleft}
