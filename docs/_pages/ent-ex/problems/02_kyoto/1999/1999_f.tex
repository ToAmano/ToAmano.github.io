\item[(1)]$f(x)$は$a \leqq x \leqq b$で連続な関数とする．このとき，
\[ \frac{1}{b-a}\int_a^bf(x)dx=f(c) \hspace{1zw} a \leqq c \leqq b \]
となる$c$が存在することを示せ．
\item[(2)]$y=\sin x$の$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$の部分と$y=1$および$y$軸が囲む図形を，
$y$軸のまわりに回転して得られる立体を考える．
この立体を$y$軸に垂直な$n-1$個の平面によって各部分の体積が等しくなるように$n$個に分割するとき，
$y=1$に最も近い平面の$y$座標を$y_n$とする．
このとき，$\displaystyle\lim_{n\to\infty}n(1-y_n)$を求めよ．
\end{description}
