　$f(x)=ax^3+bx^2+cx$を$x$の3次式とする．
すべての整数$n$に対して$f(n)$が整数になるための必要十分条件は，適当な整数$p$，$q$，$r$をとると，
\[ f(x)=\frac{p}{6}x(x+1)(x+2)+\frac{q}{2}x(x+1)+rx \]
と表されることであることを示せ．
