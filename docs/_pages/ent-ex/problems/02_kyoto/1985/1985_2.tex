　平面上の3点$A$，$B$，$C$と1次変換$f$について，下の3条件\ding{"AC}，\ding{"AD}，\ding{"AE}を仮定する．
\begin{description}
\item[\ding{"AC}]$A$，$B$，$C$は同一直線上にはなく，また原点$O$は三角形$ABC$の内部には属さない．
\item[\ding{"AD}]3点$A$，$B$，$C$の$f$による像は，全体として，3点$A$，$B$，$C$に一致する，
すなわち$ \{ f(A),f(B),f(C) \} = \{ A,B,C \} $
\item[\ding{"AE}]$f$は恒等変換ではない，すなわち$f \neq E$
\end{description}
このとき，3点$A$，$B$，$C$のうち$f$によって動かないものは，1つあって，1つに限ることを示せ．
\end{flushleft}
