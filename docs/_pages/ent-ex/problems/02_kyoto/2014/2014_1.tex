{\huge 1}
　座標空間における次の3つの直線$l$，$m$，$n$を考える：
\begin{description}
\item[]$l$は点$A(1, \, 0, \, -2)$を通り，
ベクトル$\vec{u}=(2, \, 1, \, -1)$に平行な直線である．
\item[]$m$は点$B(1, \, 2, \, -3)$を通り，
ベクトル$\vec{v}=(1, \, -1, \, 1)$に平行な直線である．
\item[]$n$は点$C(1, \, -1, \, 0)$を通り，
ベクトル$\vec{w}=(1, \, 2, \, 1)$に平行な直線である．
\end{description}
$P$を$l$上の点として，$P$から$m$，$n$へ下ろした垂線の足をそれぞれ$Q$，$R$とする．
このとき，${PQ}^2+{PR}^2$を最小にするような$P$と，そのときの${PQ}^2+{PR}^2$を求めよ．
\end{flushleft}
