{\huge 2}
　2つの粒子が時刻0において$\triangle ABC$の頂点$A$に位置している．
これらの粒子は独立に運動し，それぞれ1秒ごとに隣の頂点に等確率で移動していくとする．
たとえば，ある時刻で点$C$にいる粒子は，その1秒後には点$A$または点$B$に
それぞれ$\displaystyle\frac{1}{2}$の確率で移動する．
この2つの粒子が，時刻0の$n$秒後に同じ点にいる確率$p(n)$を求めよ．
\end{flushleft}
