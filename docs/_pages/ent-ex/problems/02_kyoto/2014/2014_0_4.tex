{\huge 4}
　次の式
\[ a_1=2, \hspace{1zw} a_{n+1}=2a_n-1 \hspace{1zw} (n=1, \, 2, \, 3, \, \cdots) \]
で定められる数列$\{ a_n \}$を考える．
\begin{description}
\item[(1)]数列$\{ a_n \}$の一般項を求めよ．
\item[(2)]次の不等式
\[ {a_n}^2-2a_n>{10}^{15} \]
を満たす最小の自然数$n$を求めよ．
ただし，$0.3010<\log_{10}2<0.3011$であることは用いてよい．
\end{description}
\end{flushleft}
