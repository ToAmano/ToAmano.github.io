{\huge 5}
　三角形$ABC$において，角$B$の二等分線と角$C$の外角の二等分線との交点を$P$とし，
また，角$C$の二等分線と角$B$の外角の二等分線との交点を$Q$とする．
二点$P$，$Q$から，辺$BC$の延長におろした垂線の足を，それぞれ$H$，$K$とする．
このとき，$KB=CH$となることを証明せよ．
\end{flushleft}
