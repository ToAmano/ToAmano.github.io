{\huge 3}
　$n$個の自然数$a_1,a_2,\cdots,a_n$を，次の2条件(1)，(2)を同時にみたすように選ぶとき，選び方は何通りあるか．
\begin{description}
\item[(1)]どの$a_i$も$1,2,\cdots,m$ $(m>n)$の中から選ぶ．
\item[(2)]$a_1,a_2,\cdots,a_n$のうち，適当な$n-1$個を選べば，それらは互いに異なる．
\end{description}
