　$xyz$空間で，$(0,0,1)$を中心とする半径1の球面を$S_1$，$(0,1,0)$を中心とする半径1の球面を$S_2$とする．

点$P$を中心とする半径$\displaystyle\frac{1}{2}$の球面を$S_0$とする．
$S_0$が平面$x-z=0$と交わってできる円の半径が$\displaystyle\frac{\sqrt2}{4}$であるとき，
$a$，$b$，$c$の値を求めよ．
