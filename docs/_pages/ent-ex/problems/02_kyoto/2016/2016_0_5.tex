{\huge 5}
　実数を係数とする3次式$f(x)=x^3+ax^2+bx+c$に対し，次の条件を考える．
\begin{description}
\item[（イ）]方程式$f(x)=0$の解であるすべての複素数$\alpha$に対し，
$\alpha^3$もまた$f(x)=0$の解である．
\item[（ロ）]方程式$f(x)=0$は虚数解を少なくとも1つもつ．
\end{description}
この2つの条件（イ），（ロ）を同時に満たす3次式をすべて求めよ．
\end{flushleft}
