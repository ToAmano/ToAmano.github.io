{\huge 4}
　座標空間に3点$P$，$Q$，$R$があって毎秒1の速さで，それぞれ
\begin{description}
\item[]点$P$は原点$(0,0,0)$を出発して$x$軸上を正の方向へ，
\item[]点$Q$は点$(2,0,0)$を出発して$y$軸と平行に正の方向へ，
\item[]点$R$は点$(2,2,0)$を出発して$z$軸と平行に正の方向へ
\end{description}
進む．このとき
\begin{description}
\item[(1)]三角形$PQR$は常に二等辺三角形であることを示せ．
\item[(2)]三角形$PQR$の面積$S$が最小となるのは何秒後か．
\end{description}
\end{flushleft}
