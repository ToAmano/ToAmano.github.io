　$f(x)={(x-1)}^3+x$，$f_1(x)=f(x)$とし，$n\geqq2$に対して，$f_n(x)=f(f_{n-1}(x))$とする．
\begin{description}
\item[(1)]$y=f(x)$のグラフをかけ．
\item[(2)]どんな$n$ $(\geqq1)$についても，$f_n(x)=x$の解は$x=1$に限ることを示せ．
\end{description}
