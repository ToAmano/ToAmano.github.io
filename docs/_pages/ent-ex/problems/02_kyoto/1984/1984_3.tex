　実数$t$の値によって定まる点$P(t+1,t)$と$Q(t-1,-t)$がある．
\begin{description}
\item[(1)]$t$がすべての実数を動くとき，直線$PQ$が通過する範囲を図示せよ．
\item[(2)]$t$が区間$[0,1]=\{t|0 \leqq t \leqq1\}$を動くとき，線分$PQ$が通過する範囲の面積を求めよ．
\end{description}
