{\huge 6}
　区間$[a,b]=\{ x|a \leqq x \leqq b \}$で定義された関数$f(x)$について，次のことを仮定する．
\begin{description}
\item[(イ)]関数$f(x)$は連続であり，更に$f(x)$およびその導関数$f'(x)$について，
微分と積分を自由に行うことができる．
\item[(ロ)]区間$[a,b]$で，つねに不等式$a \leqq f(x) \leqq b$が成り立つ．
\item[(ハ)]区間$[a,b]$でつねに不等式$|f'(x)|\leqq k<1$が成り立つような定数$k$が存在する．
\end{description}
　このとき，次の問に答えよ．
\begin{description}
\item[(1)]等式$f(c_0)=c_0$を満たす値$c_0$が，区間$[a,b]$の中に，ただ1つ存在することを，簡単に説明せよ．
\item[(2)]区間$[a,b]$の中の(任意の)1つの値$d_0$から出発して，
$d_1=f(d_0)$，$d_{n+1}=f(d_n)$ $(n=1,2,\cdots)$とおく．
このとき，不等式$|d_n-c_0| \leqq k^n|d_0-c_0|$ $(n=1,2,\cdots)$が成立することを示せ．
ただし，$c_0$は問(1)に述べられている値とする．
\end{description}
\end{flushleft}
