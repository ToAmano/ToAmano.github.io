　$f(x)$と$g(x)$はいずれも$x$の整式で，次の条件を満たしているとする．
\begin{description}
\item[(イ)]$f(x)={ \{ g(x) \} }^2+a$(ただし$a$はある実数．)
\item[(ロ)]$g'(x)=x+1$(ただし$g'(x)$は$g(x)$の導関数を表す．)
\item[(ハ)]$f(-1)=f(1)=0$
\end{description}
　実数$k$を定数とするとき，$y=f(x)$のグラフと$y=g(x)+k$のグラフとの($k$の値によって変わる)交点の個数を求めよ．
\end{flushleft}
