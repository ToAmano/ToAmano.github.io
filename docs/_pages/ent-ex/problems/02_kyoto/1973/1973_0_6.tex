{\huge 6}
　放物線$y=ax^2-4$ $(a>0)$と直線$y=5$とで囲まれた部分の面積を$S$とし，
その部分を$y$軸のまわりに一回転してできる回転体の体積を$V$とする．\\
　$S$，$V$の値を求め，$S$と$V$の間の$a$を含まない関係式を導け．
\end{flushleft}
