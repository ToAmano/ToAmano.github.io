{\huge 4}
　負でない実数$a$に対し，$0 \leqq r < 1$で，$a-r$が整数となる実数$r$を$\{ a \}$で表す．
すなわち，$\{ a \}$は，$a$の小数部分を表す．
\begin{description}
\item[(1)]$\{ n\log_{10}2 \} < 0.02$となる正の整数$n$を1つ求めよ．
\item[(2)]10進法による表示で$2^n$の最高位の数字が7となる正の整数$n$を1つ求めよ．
ただし，$0.3010<\log_{10}2<0.3011$，$0.8450<\log_{10}7<0.8451$である．
\end{description}
\end{flushleft}
