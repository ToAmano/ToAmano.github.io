{\huge 5}
　青玉$a$個，赤玉$b$個，白玉$c$個，合計$N=a+b+c$個の玉が入っている袋がある．
この袋から無作為に1個の玉を取り出し，色を見て袋にもどす．
これを$n$回繰り返す．
取り出される玉の色の数の期待値を$E_n$とするとき，
$\displaystyle E_n=3-
{\left(\frac{a+b}{N}\right)}^n-{\left(\frac{b+c}{N}\right)}^n-{\left(\frac{c+a}{N}\right)}^n$を示せ．
\end{flushleft}
