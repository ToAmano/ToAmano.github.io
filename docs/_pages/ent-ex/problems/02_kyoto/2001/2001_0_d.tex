{\huge 4}
　複素数平面上の単位円に内接する正五角形で，1がその頂点の1つとなっているものを考える．
この正五角形の辺を延長してできる直線の交点のうち，もとの正五角形の頂点以外のもので，実部，虚部がともに正であるものを$z$とする．
\begin{description}
\item[(1)]$\alpha=\cos{72}^\circ+i\sin{72}^\circ$とするとき，$\alpha$を用いて$z$を表せ．
ただし，$i$は虚数単位を表す．
\item[(2)]3点1，$\alpha^2$，$z$を通る円は，原点を通ることを示せ．
\end{description}
