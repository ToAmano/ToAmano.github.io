{\huge 3}
　$A_1$，$A_2$，$A_3$は$xy$平面上の点で同一直線上にはないとする．3つの一次式
\[ f_1(x,y)=a_1x+b_1y+c_1, \hspace{1zw} f_2(x,y)=a_2x+b_2y+c_2, \hspace{1zw} f_3(x,y)=a_3x+b_3y+c_3 \]
は，方程式$f_1(x,y)=0$，$f_2(x,y)=0$，$f_3(x,y)=0$により
それぞれ直線$A_2A_3$，$A_3A_1$，$A_1A_2$を表すとする．
このとき実数$u$，$v$をうまくとると方程式
\[ uf_1(x,y)f_2(x,y)+vf_2(x,y)f_3(x,y)+f_3(x,y)f_1(x,y)=0 \]
が3点$A_1$，$A_2$，$A_3$を通る円を表すようにできることを示せ．
\end{flushleft}
