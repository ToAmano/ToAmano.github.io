　$n$と$k$を自然数とし，整式$x^n$を整式$(x-k)(x-k-1)$で割った余りを$ax+b$とする．
\begin{description}
\item[(1)]$a$と$b$は整数であることを示せ．
\item[(2)]$a$と$b$をともに割り切る素数は存在しないことを示せ．
\end{description}
