　$-1 \leqq x \leqq 1$で定義された関数$y=f(x)$はつぎの\ding{"AC}，\ding{"AD}を満たしている．
\begin{description}
\item[\ding{"AC}]$\sin f(x)=1-x^2$
\item[\ding{"AD}]$\displaystyle 0 \leqq f(x) \leqq \frac{\pi}{2}$
\end{description}
\begin{description}
\item[(1)]$x$を$y$の関数として表し，$y=f(x)$のグラフの概形をかけ．
\item[(2)]$y=f(x)$のグラフと$x$軸で囲まれた部分の面積を求めよ．
\end{description}
\end{flushleft}
