　$a$，$b$は正の実数とする．
円$C:x^2+{(y-a)}^2=1$，双曲線$H:y^2-x^2=b^2$を考える．
$H$の上半分($y$座標が正の部分)を$H_+$，下半分($y$座標が負の部分)を$H_-$とする．
\begin{description}
\item[(1)]$C$が$H_+$と相異なる2点を共有し，かつ$H_-$とも相異なる2点を共有するような$(a,b)$の範囲を図示せよ．
\item[(2)]$a$，$b$を適当にとれば，$C$が$H_{+}$と相異なる4点を共有するようにできるか否か．理由をつけて答えよ．
\end{description}
\end{flushleft}
