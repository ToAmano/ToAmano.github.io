\item[(i)]　次の□の中に適当な数あるいは語句をいれ，その理由をしるせ．\\
　$x>0$において，直線$y=ax$ $(a\text{は正数})$と曲線$xy=1$の交点を$P$とする．
原点を$O$とするとき，線分$OP$の長さ$\overline{OP}$が最小となるのは$a=\text{□}^{ア)}$のときで，
そのとき$\overline{OP}=\text{□}^{イ)}$である．
$a\geqq1$では，$a$が増加するとともに$\overline{OP}$は$\text{□}^{ウ)}$し，
$a<1$では，$a$が増加するとともに$\overline{OP}$は$\text{□}^{エ)}$する．
\item[(ii)]$0<xy\leqq1$，$\displaystyle1\leqq\frac{y}{x}\leqq\sqrt3$，$x>0$をみたす点$(x,y)$の範囲を考える．
点$Q$はこの範囲にあるものとして，$\overline{OQ}$の最大値，およびそのとき線分$OQ$が$x$軸となす角を求めよ．
