{\huge 3}
　台形$ABCD$の平行な2辺$AB$，$DC$上にそれぞれ点$P$，$Q$をとる．
次に$AB$に平行な直線が辺$AD$，$BC$および線分$PQ$と交わる点をそれぞれ$L$，$M$，$N$とし，
$AP=a$，$PB=b$，$DQ=c$，$QC=d$，$LN=x$，$NM=y$とする．
$ad-bc>0$，$c\neq0$のとき，
$\displaystyle\frac{y}{x}$，$\displaystyle\frac{b}{a}$，$\displaystyle\frac{d}{c}$の大小をしらべよ．
