{\huge 5}
　0以上の整数を10進法で表すとき，次の問いに答えよ．
ただし，0は0桁の数と考えることにする．
また$n$は正の整数とする．
\begin{description}
\item[(1)]各桁の数が1または2である$n$桁の整数を考える．
それらすべての整数の総和を$T_n$とする．
$T_n$を$n$を用いて表せ．
\item[(2)]各桁の数が0，1，2のいずれかである$n$桁以下の整数を考える．
それらすべての整数の総和を$S_n$とする．
$S_n$が$T_n$の15倍以上になるのは，$n$がいくつ以上のときか．
必要があれば，$0.301<\log_{10}2<0.302$および
$0.477<\log_{10}3<0.478$を用いてもよい．
\end{description}
\end{flushleft}
