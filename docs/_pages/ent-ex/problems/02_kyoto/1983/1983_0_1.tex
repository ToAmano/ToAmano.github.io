　数直線上を原点から右(正の向き)に硬貨を投げて進む．
表が出れば1進み，裏が出れば2進むものとする．
このようにして，ちょうど点$n$に到達する確率を$p_n$で表す．
ただし，$n$は自然数とする．
\begin{description}
\item[(1)]3以上の$n$について，$p_n$と$p_{n-1}$，$p_{n-2}$との関係式を求めよ．
\item[(2)]$p_n$ $(n\geqq3)$を求めよ．
\end{description}
