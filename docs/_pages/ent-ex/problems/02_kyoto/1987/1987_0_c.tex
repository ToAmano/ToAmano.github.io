　$a$，$b$は正の定数で，平面上に4点$O(0,0)$，$A(0,a)$，$B(1,b)$，$C(1,0)$と，

\begin{description}

\item[(2)]$H$の弧$\stackrel{\frown}{AB}$が台形$OABC$の面積を二等分することはないことを示せ．
\end{description}
