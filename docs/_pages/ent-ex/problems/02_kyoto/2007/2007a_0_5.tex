{\huge 5}
　$n$を1以上の整数とするとき，次の2つの命題はそれぞれ正しいか．
正しいときは証明し，正しくないときはその理由を述べよ．\\
命題$\mathbf{p}$：ある$n$に対して，$\sqrt{n}$と$\sqrt{n+1}$は共に有理数である．\\
命題$\mathbf{q}$：すべての$n$に対して，$\sqrt{n+1}-\sqrt{n}$は無理数である．
\end{flushleft}
