{\huge 4}
　正の実数$a$に対し，
実数全体で定義される関数$g(x)$を$\displaystyle g(x)=\int_{-2}^2|x-t|(t^2-a^2)dt$で定める．
このとき，
$g(x)$が最小値を持つような$a$の範囲を求めよ．
また$a$がそのような範囲にあるとき，
$g(x)$の最小値を$a$を用いて表せ．
\end{flushleft}
