{\huge 3}
　次の文中の□に，適当な語句または文を入れよ．\\
　1つの命題「$A$ならば$B$である」について，条件$A$をみたすものが存在しないならば，この命題は□．
その理由は，次の通りである．
1つの命題と，その□とは互いに同値であるから，次の命題を考えればよい．
\begin{center}
「$B$で□ならば，$A$で□．」
\end{center}
　この命題の結論「$A$で□」は□から，この命題は□．
ゆえに，もとの命題「$A$ならば$B$である」は□．\\
　上のようなことが実際にあてはまる例を考えよう．次の命題を$P$と呼ぼう．
\begin{center}
「$x$，$y$が実数であって，$x^2+y^2+1=0$であるならば，$x=0$または$y=0$である．」
\end{center}
　この$P$を，「$A$ならば$B$である」の形にして考えると，\\
$A$は「□」であり，$B$は「□」である．\\
$A$の否定は「□」であり，$B$の否定は「□」である．\\
　したがって，$P$の逆，対偶は，それぞれ，次の通りである．
\begin{center}
逆:□\\
対偶:□
\end{center}
　$P$，$P$の逆，$P$の対偶のうち，
(イ)初めに述べた命題の例になっているものは□であり，
(ロ)真であるものは□である．
\end{flushleft}
