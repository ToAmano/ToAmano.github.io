{\huge 3}
　関数$f(x)$が次の2つの性質(1)，(2)を持つという．
\begin{description}
\item[(1)]任意の実数$x$，$y$に対して，$f(x+y)=f(x)f(y)$が成り立つ．
\item[(2)]$f(3)=8$
\end{description}
このとき，$f(1)=2$であることを証明せよ．(ただし，$f(x)$は実数であるとする．)
\end{flushleft}
