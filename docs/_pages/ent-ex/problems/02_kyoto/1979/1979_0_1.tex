　平面上に6つの定点$A_1$，$A_2$，$A_3$，$A_4$，$A_5$，$A_6$があって，どの3点も一直線上にはない．
この6点のうちから3点を任意にえらぶ．
えらんだ3点を頂点とする三角形の重心と，残りの3点を頂点とする三角形の重心とを通る直線は，
3点のえらびかたに無関係な一定の点を通ることを示せ．
