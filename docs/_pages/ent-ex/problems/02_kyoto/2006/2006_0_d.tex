{\huge 4}
　$n$を自然数とし，$xy$平面の次の領域
$\displaystyle D_n=\left\{ (x,y) \Bigg| \frac{x}{\displaystyle n+\frac{1}{2}} \leqq y \leqq [x+1]-x,x\geqq0 \right\}$
を考える．
ただし，記号$[x]$は$x$より大きくない最大の整数を表すものとする．
このとき$D_n$の面積を求めよ．
