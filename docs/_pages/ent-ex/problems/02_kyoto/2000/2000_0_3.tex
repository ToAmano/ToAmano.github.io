{\huge 3}
　$\vec{a}=(1,0,0)$，$\vec{b}=(\cos{60}^\circ,\sin{60}^\circ,0)$とする．
\begin{description}
\item[(1)]長さ1の空間ベクトル$\vec{c}$に対し，$\cos\alpha=\vec{a}\cdot\vec{c}$，$\cos\beta=\vec{b}\cdot\vec{c}$とおく．
このとき次の不等式(＊)が成り立つことを示せ．
\begin{description}
\item[(＊)]$\displaystyle \cos^2\alpha-\cos\alpha\cos\beta+\cos^2\beta\leqq\frac{3}{4}$
\end{description}
\item[(2)]不等式(＊)を満たす$(\alpha,\beta)$ $(0\leqq\alpha\leqq{180}^\circ,0\leqq\beta\leqq{180}^\circ)$の
範囲を図示せよ．
\end{description}
\end{flushleft}
