{\huge 4}
　関数$f(x)=x^3+ax^2+bx+c$は次の条件(イ)，(ロ)を満たしている．
\begin{description}
\item[(イ)]$y=f(x)$のグラフは，点$(0,1)$に関して点対称である．
\item[(ロ)]$y=f(x)$は相異なる2つの極値をもち，2つの極値の差の絶対値は4に等しい．
\end{description}
　このとき
\begin{description}
\item[(1)]$y=f(x)$のグラフは$x$軸と相異なる3点で交わることを示せ．
\item[(2)](1)における3点の$x$座標を$\alpha$，$\beta$，$\gamma$(ただし$\alpha<\beta<\gamma$とする)とおくとき，
$\displaystyle f \left( \frac{-\beta-\gamma}{2} \right) >2$を示せ．
\end{description}
\end{flushleft}
