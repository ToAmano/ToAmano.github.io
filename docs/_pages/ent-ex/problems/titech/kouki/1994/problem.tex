\section{問題1}
関数 $f(x)$ に対し
\[ F(x) = \int_0^x f(t)dt \]
とおく．ある定数 $a, b, c$ が存在して
\[ F(x) = x^2 + a|x-b| + cx \]
が常に成立し，さらに $3$ つの条件
\begin{enumerate}
    \item[(i)] $f(x)$ は連続
    \item[(ii)] $F(1) = 0$
    \item[(iii)] $f(0) = 1$
\end{enumerate}
が満たされているとする．このとき $f(x)$ を求めよ．
