{\huge 3}
　4角錐$V-ABCD$があって，その底面$ABCD$は正方形であり，また4辺$VA$，$VB$，$VC$，$VD$の長さはすべて相等しい．
この4角錐の頂点$V$から底面に下した垂線$VH$の長さは6であり，底面の1辺の長さは$4\sqrt3$である．
$VH$上に$VK=4$なる点$K$をとり，点$K$と底面の1辺$AB$とを含む平面で，この4角錐を2つの部分に分けるとき，
頂点$V$を含む部分の体積を求めよ．
