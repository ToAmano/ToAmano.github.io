{\huge 4}
　$r$を0以上の整数とし，数列$\{ a_n \}$を次のように定める。
\[ a_1=r, \hspace{1zw} a_2=r+1, \hspace{1zw} a_{n+2}=a_{n+1}(a_n+1) \hspace{1zw}
(n=1, \, 2, \, 3, \, \cdots) \]
また，素数$p$を1つとり，$a_n$を$p$で割った余りを$b_n$とする。
ただし，0を$p$で割った余りは0とする。
\begin{description}
\item[(1)]自然数$n$に対し，$b_{n+2}$は$b_{n+1}(b_n+1)$を$p$で割った余りと一致することを示せ。
\item[(2)]$r=2$，$p=17$の場合に，10以下のすべての自然数$n$に対して，$b_n$を求めよ。
\item[(3)]ある2つの相異なる自然数$n$，$m$に対して，
\[ b_{n+1}=b_{m+1}>0, \hspace{1zw} b_{n+2}=b_{m+2} \]
が成り立ったとする。このとき，$b_n=b_m$が成り立つことを示せ。
\end{description}
\end{flushleft}
