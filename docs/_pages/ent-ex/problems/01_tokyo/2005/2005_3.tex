　関数$f(x)$を$\displaystyle f(x)=\frac{1}{2}x \{ 1+e^{-2(x-1)} \}$とする．
ただし，$e$は自然対数の底である．
\begin{description}
\item[(1)]$\displaystyle x>\frac{1}{2}$ならば$\displaystyle 0 \leqq f'(x) < \frac{1}{2}$であることを示せ．
\item[(2)]$x_0$を正の数とするとき，数列$\{ x_n \}$ $(n=0,1,\cdots)$を，$x_{n+1}=f(x_n)$によって定める．

$\displaystyle x_0>\frac{1}{2}$であれば，$\displaystyle \lim_{n\to\infty}x_n=1$であることを示せ．
\end{description}
\end{flushleft}
