{\huge 3}
　座標空間内に5点$A(2, \, 0, \, 0)$，$B(0, \, 2, \, 0)$，$C(-2, \, 0, \, 0)$，
$D(0, \, -2, \, 0)$，$E(0, \, 0, \, -2)$を考える。
線分ABの中点Mと線分ADの中点Nを通り，直線AEに平行な平面を$\alpha$とする。
さらに，$p$は$2<p<4$をみたす実数とし，点$P(p, \, 0, \, 2)$を考える。
\begin{description}
\item[(1)]八面体PABCDEの平面$y=0$による切り口および，
平面$\alpha$の平面$y=0$による切り口を同一平面上に図示せよ。
\item[(2)]八面体PABCDEの平面$\alpha$による切り口が八角形となる$p$の範囲を求めよ。
\item[(3)]実数$p$が(2)で定まる範囲にあるとする。
八面体PABCDEの平面$\alpha$による切り口のうち
$y\geqq0$，$z\geqq0$の部分を点$(x, \, y, \, z)$が動くとき，
座標平面上で点$(y, \, z)$が動く範囲の面積を求めよ。
\end{description}
\end{flushleft}
