点$(s,t)$から放物線$C$へ引いた接線を$l_1$，$l_2$とする。
\item[(1)]$l_1$，$l_2$の方程式を求めよ。
\item[(2)]$a$を正の実数とする。
放物線$C$と直線$l_1$，$l_2$で囲まれる領域の面積が$a$となる$(s,t)$を全て求めよ。
