{\huge 3}
　2つの関数
\[ f(x)=ax^3+bx^2+cx \]
\[ g(x)=px^3+qx^2+rx \]
が次の5つの条件を満たしているとする．
\[ f'(0)=g'(0), \hspace{1zw} f(-1)=-1, \hspace{1zw} f'(-1)=0, \hspace{1zw} g(1)=3, \hspace{1zw} g'(1)=0 \]
ここで$f(x)$，$g(x)$の導関数をそれぞれ$f'(x)$，$g'(x)$で表している．\\
　このような関数のうちで，定積分
\[ \int_{-1}^0 { \{ f''(x) \} }^2dx+\int_0^1 { \{ g''(x) \} }^2dx \]
の値を最小にするような$f(x)$と$g(x)$を求めよ．
ただし，$f''(x)$，$g''(x)$はそれぞれ$f'(x)$，$g'(x)$の導関数を表す．
\end{flushleft}
