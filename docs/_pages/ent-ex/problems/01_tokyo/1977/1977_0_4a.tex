{\huge 4 新}
　座標の定められた空間において，
直線$l$は2点$(1,1,0)$，$(2,1,1)$を通り，
直線$m$は2点$(1,1,1)$，$(1,3,2)$を通る．
\begin{description}
\item[i)]$l$を含み$m$に平行な平面の方程式を$ax+by+cz+d=0$の形に表せ．
\item[ii)]点$(2,0,1)$を通り$l$，$m$の両方と交わる直線を$n$とする．
$l$と$n$の交点および$m$と$n$の交点を求めよ．
\end{description}
\end{flushleft}
