　$a\geqq1$とする．
$xy$平面において，
不等式$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$，$1 \leqq y \leqq a\sin x$
によって定められる領域の面積を$S_1$，
不等式$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$，$0 \leqq y \leqq a\sin x$，$0 \leqq y \leqq 1$
によって定められる領域の面積を$S_2$とする．
$S_2-S_1$を最大にするような$a$の値と，$S_2-S_1$の最大値を求めよ．
