{\huge 2}
　正整数$l$を与える．各正整数$n$に対して，関数
\[ y=x^l\sin nx, \hspace{1zw} 0 \leqq x \leqq 2\pi \]
のグラフと$x$軸で囲まれる図形を$C_n$とする．
\begin{description}
\item[(1)]$C_n$を$x$軸のまわりに回転させてできる回転体の体積を$V_n$とするとき，極限値
\[ \lim_{n\to\infty}V_n \]
を求めよ．
\item[(2)]$C_n$を$y$軸のまわりに回転させてできる回転体の体積を$W_n$とするとき，極限値
\[ \lim_{n\to\infty}W_n \]
を求めよ．
\end{description}
\end{flushleft}
