{\huge 3}
　与えられた実数係数の整式$f(x)$について
\[ \int_0^1 f(x)dx=2, \hspace{1zw} \int_0^1 xf(x)dx=3 \]
となるとする．そのとき，
\[ \int_0^1 { \{ f(x)-ax-b \} }^2 dx \]
の値を最小にする実数$a$および$b$の値を求めよ．
