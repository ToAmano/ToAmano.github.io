　3以上の奇数$n$に対して，$a_n$と$b_n$を次のように定める．
\[ a_n=\frac{1}{6}\sum_{k=1}^{n-1}(k-1)k(k+1), \hspace{1zw} b_n=\frac{n^2-1}{8} \]
\begin{description}
\item[(1)]$a_n$と$b_n$はどちらも整数であることを示せ．
\item[(2)]$a_n-b_n$は4の倍数であることを示せ．
\end{description}
\end{flushleft}
