　$a>1$とし，次の不等式を考える．
\[ \text{（＊）} \hspace{1zw} \frac{e^t-1}{t}\geqq e^\frac{t}{a} \]
\begin{description}
\item[(1)]$a=2$のとき，すべての$t>0$に対して上の不等式（＊）が成り立つことを示せ．
\item[(2)]すべての$t>0$に対して上の不等式（＊）が成り立つような$a$の範囲を求めよ．
\end{description}
