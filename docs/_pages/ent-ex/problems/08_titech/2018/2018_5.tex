{\huge 5}
　$xyz$空間内の一辺の長さが1の立方体
\[ \{ (x, \, y, \, z) \, | \, 0 \leqq x \leqq 1, \hspace{1zw}
0 \leqq y \leqq 1, \hspace{1zw} 0 \leqq z \leqq 1 \} \]
を$Q$とする．点$X$は頂点$A(0, \, 0, \, 0)$から出発して
$Q$の辺上を1秒ごとに長さ1だけ進んで隣の頂点に移動する．
$X$が$x$軸，$y$軸，$z$軸に平行に進む確率はそれぞれ$p$，$q$，$r$である．ただし
\[ p\geqq0, \hspace{1zw} q\geqq0, \hspace{1zw} r\geqq0,
\hspace{1zw}\hspace{1zw} p+q+r=1 \]
である．$X$が$n$秒後に頂点$A(0, \, 0, \, 0)$，$B(1, \, 1, \, 0)$，
$C(1, \, 0, \, 1)$，$D(0, \, 1, \, 1)$にある確率を
それぞれ$a_n$，$b_n$，$c_n$，$d_n$とする．
\begin{description}
\item[(1)]$a_{n+2}$を$a_n$，$b_n$，$c_n$，$d_n$と$p$，$q$，$r$を用いて表せ．
\item[(2)]$a_n-b_n+c_n-d_n$を$p$，$q$，$r$，$n$を用いて表せ．
\item[(3)]$a_n$を$p$，$q$，$r$，$n$を用いて表せ．
\end{description}
\end{flushleft}
