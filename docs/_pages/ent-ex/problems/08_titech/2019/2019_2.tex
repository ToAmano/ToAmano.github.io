　次の等式が$1\leqq x \leqq2$で成り立つような関数$f(x)$と定数$A$，$B$を求めよ．
\[ \int_\frac{1}{x}^\frac{2}{x}|\log y|f(xy)dy=3x(\log x-1)+A+\frac{B}{x} \]
　ただし，$f(x)$は$1\leqq x \leqq2$に対して定義される連続関数とする．
