{\huge 3}
　$\triangle ABC$において，辺$AB$の中点を$M$，辺$AC$の中点を$N$とする．
辺$AB$を$x:1-x$ $(0 \leqq x < 1)$の比に内分する点$P$と，辺$AC$を$y:1-y$ $(0 \leqq y < 1)$の比に内分する点$Q$をとり、
線分$BQ$と線分$CP$の交点を$R$とする．
このとき，$R$が$\triangle AMN$に含まれるような$(x,y)$全体を$xy$平面に図示し，その面積を求めよ．
(ただし，辺$AB$，辺$AC$を$0:1$の比に内分する点とは，ともに点$A$のこととする．)
\end{flushleft}
