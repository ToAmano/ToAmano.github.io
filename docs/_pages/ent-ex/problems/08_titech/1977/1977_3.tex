　放物線$y=x^2$上の点$P(x, \, y)$と
$x$軸上の点$\displaystyle A \left( -\frac{1}{3}, \, 0 \right)$を考える．
ただし$x>0$とする．
点$P$における放物線の接線が直線$AP$および$y$軸と等しい角度で交わるとき，
点$P$の座標を求めよ．
