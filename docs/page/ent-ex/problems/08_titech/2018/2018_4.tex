{\huge 4}
　$xyz$空間内において，連立不等式
\[ \frac{x^2}{4}+y^2\leqq1, \hspace{1zw} |z|\leqq6 \]
により定まる領域を$V$とし，
2点$(2, \, 0, \, 2)$，$(-2, \, 0, \, -2)$を通る直線を$l$とする．
\begin{description}
\item[(1)]$|t|\leqq2\sqrt2$を満たす実数$t$に対し，
点$\displaystyle P_t \left(\frac{t}{\sqrt2}, \, 0, \, \frac{t}{\sqrt2}\right)$を通り
$l$に垂直な平面を$H_t$とする．
また，実数$\theta$に対し，点$(2\cos\theta, \, \sin\theta, \, 0)$を通り
$z$軸に平行な直線を$L_\theta$とする．
$L_\theta$と$H_t$との交点の$z$座標を$t$と$\theta$を用いて表せ．
\item[(2)]$l$を回転軸に持つ回転体で$V$に含まれるものを考える．
このような回転体のうちで体積が最大となるものの体積を求めよ．
\end{description}
\end{flushleft}
