{\huge 2}
　楕円$\displaystyle C:\frac{x^2}{a^2}+y^2=1$ $(a \geqq 1)$が与えられている．
\begin{description}
\item[(1)]$C$の外部の点$P(X,Y)$から$C$への2接線が直交するように$P$を動かす．$P$の軌跡を求めよ．
\item[(2)]$S$を(1)で求めた$P$の軌跡とする．
$S$と$C$で囲まれた部分を直線$x=2a$を軸として，回転してできる回転体の体積を求めよ．
\end{description}
\end{flushleft}
