{\huge 3}
　$n$を自然数とする．
\begin{description}
\item[(1)]$\displaystyle f(x)=\frac{x^2}{n^2}+e^{2x}-1$の増減を調べ，グラフの概形をえがけ，
\item[(2)]楕円$\displaystyle \frac{x^2}{n^2}+n^2 y^2=1$と曲線$\displaystyle y=\frac{1}{n}e^x$の交点のうち
$\displaystyle \left( 0,\frac{1}{n} \right)$でない方の座標を$(x_n,y_n)$とおく．
このとき$\displaystyle\lim_{n\to\infty}\frac{x_n}{n}=-1$であることを示せ．
\end{description}
\end{flushleft}
