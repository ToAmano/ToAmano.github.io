　次の問いに答えよ．
\begin{description}
\item[(1)]$f(x)$，$g(x)$を連続な偶関数，$m$を正の整数とするとき，
$\displaystyle \int_0^{mx}f(\sin x)g(\cos x)dx=m\int_0^{\pi}f(\sin x)g(\cos x)dx$を証明せよ．
\item[(2)]正の整数$m$，$n$が$m\pi \leqq n < (m+1)\pi$を満たしているとき，
\[ \frac{m}{(m+1)\pi}\int_0^{\pi}\frac{\sin x}{(1+\cos^2 x)^2}dx \leqq 
\int_0^1\frac{|\sin nx|}{(1+\cos^2 nx)^2}dx \leqq \frac{m+1}{m\pi}\int_0^{\pi}\frac{\sin x}{(1+\cos^2 x)^2}dx \]
を証明せよ．
\item[(3)]極限値$\displaystyle\lim_{n \to \infty}\int_0^1\frac{|\sin nx|}{(1+\cos^2 nx)^2}dx$を求めよ．
\end{description}
