{\huge 1}
　場所1から場所$n$に異なる$n$個のものが並んでいる．
これらを並べ替えてどれもが元の位置にならないようにする方法の総数を$D(n)$とする．
ただし$n \geqq 2$とする．
\begin{description}
\item[(1)]$n=4$の場合の並べ替え方をすべて書き出して，$D(4)$を求めよ．
\item[(2)]$n \geqq 4$に対して$D(n)=(n-1) \{ D(n-2)+D(n-1) \}$を証明せよ．
\end{description}
\end{flushleft}
