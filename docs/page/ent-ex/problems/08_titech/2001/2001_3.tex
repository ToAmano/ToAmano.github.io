{\huge 3}
　箱の中に1から$N$までの番号が一つずつ書かれた$N$枚のカードが入っている．
この箱から無作為にカードを1枚取り出して戻すという試行を$k$回行う．
このとき，はじめから$j$回目$(j=1,\cdots\cdots,k)$までに取り出したカードの番号の和を$X_j$とし，
$X_1,\cdots\cdots,X_k$のうちのどれかが$k$となる確率を$P_N(k)$とする．
\begin{description}
\item[(1)]$N \geqq 3$のとき$P_N(1)$，$P_N(2)$，$P_N(3)$を$N$で表せ．
\item[(2)]$P_3(4)$，$P_3(5)$を求めよ．
\item[(3)]$k \leqq N$のとき，$P_N(k)$を$N$と$k$で表せ．
\end{description}
\end{flushleft}
