　$x$，$y$，$z$は条件$0 \leqq x \leqq 1$，$0 \leqq y \leqq 1$，$2 \leqq z \leqq 3$を満たしながら変化する。
$\displaystyle w=\frac{z-x}{z-y}$とおくとき，次の問に答えよ。
\begin{description}
\item[(1)]$w=3$となることがあるか。
\item[(2)]$z$を一定にしたとき，$w$の最大値を$z$で表わせ。
\item[(3)]$w$の最大値を求めよ。
\item[(4)]$z$のどんな値に対しても$w=k$を満たす$x$，$y$が存在するような$k$の最大値を求めよ。
\end{description}
