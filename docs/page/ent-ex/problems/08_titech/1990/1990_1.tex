　$x$，$y$，$z$，$w$を正数とする．
任意の正の整数$m$，$n$に対して
\[ {(x^\frac{1}{m}+y^\frac{1}{m})}^n+{(z^\frac{1}{m}+w^\frac{1}{m})}^n=
{ \{ {(x^\frac{n}{m}+z^\frac{n}{m})}^\frac{1}{n}+{(y^\frac{n}{m}+w^\frac{n}{m})}^\frac{1}{n} \} }^n \]
が成り立つための必要十分条件を求めよ．
