{\huge 3}
　3次関数$y=x^3-3x^2+2x$のグラフを$C$，直線$y=ax$を$l$とする．
\begin{description}
\item[(1)]$C$と$l$が原点以外の共有点をもつような実数$a$の範囲を求めよ．
\item[(2)]$a$が(1)で求めた範囲内にあるとき，
$C$と$l$によって囲まれる部分の面積を$S(a)$とする．
$S(a)$が最小となる$a$の値を求めよ．
\end{description}
\end{flushleft}
