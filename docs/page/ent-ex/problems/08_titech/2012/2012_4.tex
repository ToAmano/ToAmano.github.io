{\huge 4}
　$n$を正の整数とする．
数列$ \{ a_k \} $を 
\[ a_1=\frac{1}{n(n+1)}, \hspace{1zw}
a_{k+1}=-\frac{1}{k+n+1}+\frac{n}{k}\sum_{i=1}^k a_i  \hspace{1zw}
(k=1,2,3,\cdots)\]
によって定める． 
\begin{description}
\item[(1)]$a_2$および$a_3$を求めよ．
\item[(2)]一般項$a_k$を求めよ．
\item[(3)]$b_n=\displaystyle\sum_{k=1}^n \sqrt{a_k}$とおくとき，
$\displaystyle\lim_{n \to \infty}b_n=\log2$を示せ．
\end{description}
\end{flushleft}
