　次の問に答えよ．
\begin{description}
\item[(1)]実数$a_1$，$a_2$，$x_1$，$x_2$，$y_1$，$y_2$が
\[ 0 < a_1 \leqq a_2 \]
\[ a_1x_1 \leqq a_1y_1 \]
\[ a_1x_1 + a_2x_2 \leqq a_1y_1 + a_2y_2 \]
をみたしているとする．
このとき$x_1 + x_2 \leqq y_1 + y_2$であることを証明せよ．
\item[(2)]$n$を2以上の整数とし，$3n$個の実数$a_1,a_2,\cdots,a_n,x_1,x_2,\cdots,x_n,y_1,y_2,\cdots,y_n$が
\[ 0 < a_1 \leqq a_2 \leqq \cdots \leqq a_n \]
および$n$個の不等式
\[ \sum_{i=1}^j a_ix_i \leqq \sum_{i=1}^j a_iy_i \hspace{1zw} (j=1,2,\cdots,n) \]
をみたしているならば，
\[ \sum_{i=1}^n x_i \leqq \sum_{i=1}^n y_i \]
であることを証明せよ．
\end{description}
\end{flushleft}
