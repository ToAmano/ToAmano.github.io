{\huge 2}
　$yz$平面の直線$y=z$を$l_1$，直線$y=z+\sqrt{2}$を$l_2$とする．
$xyz$空間において$l_1$を軸にして$l_2$を回転してできる円柱面(内部は含まない)を$C$とする．
さらに$z$軸を軸として$C$を回転してできる回転体を$R$とする．
\begin{description}
\item[(1)]$xy$平面で$C$を切った切り口に現れる楕円の方程式を求めよ．
\item[(2)]$R$の$yz$平面による断面を図示せよ．
\item[(3)]$R$の$-2 \leqq z \leqq 2$の部分の体積を求めよ．
\end{description}
\end{flushleft}
