{\huge 4}
　正の整数$n$に対し，
$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$の範囲において
$\sin4nx\geqq\sin x$を満たす$x$の区間の長さの総和を$S_n$とする．
このとき，$\displaystyle\lim_{n\to\infty}S_n$を求めよ．
