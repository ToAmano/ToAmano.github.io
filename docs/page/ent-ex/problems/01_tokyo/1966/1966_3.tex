　平面上に点列$P_0,P_1,P_2,\cdots,P_n,\cdots$があり，
$P_0$，$P_1$の座標はそれぞれ$(0,0)$，$(1,0)$である．
また，任意の自然数$n$に対し，
線分$P_nP_{n+1}$の長さは，線分$P_{n-1}P_n$の長さの2倍で，
半直線$P_nP_{n+1}$が半直線$P_{n-1}P_n$となす角は120$^\circ$である．
$P_{3n}$の座標を求めよ．
