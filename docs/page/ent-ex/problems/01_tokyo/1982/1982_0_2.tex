{\huge 2}
　$xy$平面上の曲線$y=x^2$上の3点を，$x$座標の小さいものから順に$A$，$B$，$C$とする．
$A$と$B$との$x$座標の差は$a$ $(a\text{は正の定数})$，
$B$と$C$との$x$座標の差は1，
という関係を保ちながら3点$A$，$B$，$C$が動く．\\
　$\angle CAB$が最大になるときの，点$A$の$x$座標を$a$で表わせ．
また，$\angle CAB$が最大になるときに，$\angle ABC$が直角になるような$a$の値を求めよ．
\end{flushleft}
