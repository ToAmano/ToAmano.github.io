　正の整数の下2\ruby{桁}{けた}とは，100の位以上を無視した数をいう．
たとえば2000，12345の下2桁はそれぞれ0，45である．
$m$が正の整数全体を動くとき，$5m^4$の下2桁として現れる数をすべて求めよ．
