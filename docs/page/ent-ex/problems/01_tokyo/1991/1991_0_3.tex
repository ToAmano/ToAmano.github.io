{\huge 3}
　二辺の長さが1と$a$の長方形の頂点$A$，$B$，$C$，$D$および対角線の共有点$E$を中心として，半径$r$の円を5つえがく．
どの2つの円の内部も共通部分をもたないようにして半径$r$を最大にするとき，5つの円が長方形から切りとる面積を$S(a)$とする．\\
　$a$の関数$\displaystyle\frac{S(a)}{a}$のグラフの概形をえがけ．
