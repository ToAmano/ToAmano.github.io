　$t$を実数の定数として，関数$f(x)=(x^2-3x+2)(x-t)$を考える．
いま$f'(x)=0$の2個の解を$\alpha$，$\beta$ $(\alpha<\beta)$と書くことにすれば，これらは$t$の関数とみなすことができる．\\
　$t$の関数$|t-\alpha|+|t-\beta|$の$1\leqq t \leqq3$の範囲における最大値および最小値を求めよ．
