　$xy$平面上に，海を隔てて2国$A$，$B$がある．
$A$の領土は不等式$x^2+{(y-7)}^2\leqq4$で表される領域であり，
$B$の領土は不等式$y\leqq0$で表される領域であるという．\\
　いま$A$の領海を次の3条件(1)，(2)，(3)を満たす点$P$全体の集合と定める:
\begin{description}
\item[(1)]$P$は$A$，$B$いずれの領土にも含まれない．
\item[(2)]$P$と$A$の領土との間の最短距離は4より小さい．
\item[(3)]$P$と$A$の領土との間の最短距離は，$P$と$B$の領土との間の最短距離より小さい．
\end{description}
$A$の領海の面積を求めよ．
