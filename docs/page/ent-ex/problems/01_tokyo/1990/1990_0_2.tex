　3次方程式$x^3+3x^2-1=0$の一つの解を$\alpha$とする．
\begin{description}
\item[(1)]${(2\alpha^2+5\alpha-1)}^2$を$a\alpha^2+b\alpha+c$の形の式で表せ．

\item[(2)]上の3次方程式の$\alpha$以外の二つの解を(1)と同じ形の式で表せ．
\end{description}
