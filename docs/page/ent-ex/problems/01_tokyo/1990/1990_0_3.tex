　$a$，$b$，$c$を整数，$p$，$q$，$r$を$p<0<q<1<r<2$をみたす実数とする．
関数$f(x)=x^4+ax^3+bx^2+c$が次の条件(i)，(ii)をみたすように$a$，$b$，$c$，$p$，$q$，$r$を定めよ．
\begin{description}
\item[(i)]$f(x)=0$は4個の相異なる実数解をもつ．
\item[(ii)]関数$f(x)$は$x=p,q,r$において極値をとる．
\end{description}
