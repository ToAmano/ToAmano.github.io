{\huge 3}
　座標平面上の2つの放物線
\begin{description}
\item[]$A:y=x^2$
\item[]$B:y=-x^2+px+q$
\end{description}
が点$(-1, \, 1)$で接している。ここで，$p$と$q$は実数である。
さらに，$t$を正の実数とし，放物線$B$を$x$軸の正の向きに$2t$，
$y$軸の正の向きに$t$だけ平行移動して得られる放物線を$C$とする。
\begin{description}
\item[(1)]$p$と$q$の値を求めよ。
\item[(2)]放物線$A$と$C$が囲む領域の面積を$S(t)$とする。
ただし，$A$と$C$が領域を囲まないときは$S(t)=0$と定める。$S(t)$を求めよ。
\item[(3)]$t>0$における$S(t)$の最大値を求めよ。
\end{description}
\end{flushleft}
