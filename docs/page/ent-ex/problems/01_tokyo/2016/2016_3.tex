　$a$を$1<a<3$をみたす実数とし，座標空間内の4点
$P_1(1, \, 0, \, 1)$，$P_2(1, \, 1, \, 1)$，$P_3(1, \, 0, \, 3)$，$Q(0, \, 0, \, a)$を考える。
直線$P_1Q$，$P_2Q$，$P_3Q$と$xy$平面の交点をそれぞれ$R_1, \, R_2, \, R_3$として，
三角形$R_1R_2R_3$の面積を$S(a)$とする。
$S(a)$を最小にする$a$と，そのときの$S(a)$の値を求めよ。
