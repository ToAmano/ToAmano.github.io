　平面上に1辺の長さが1の正方形$S$がある．
この平面上で$S$を平行移動して得られる正方形で，
点$P$を中心にもつものを$T(P)$とする．
このとき，
共通部分$S \cap T (P)$の面積が$\displaystyle\frac{1}{2}$以上となるような点$P$の存在範囲を図示せよ．
