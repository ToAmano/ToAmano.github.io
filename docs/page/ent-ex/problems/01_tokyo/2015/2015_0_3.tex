　$l$を座標平面上の原点を通り傾きが正の直線とする。
さらに，以下の3条件(i)，(ii)，(iii)で定まる円$C_1$，$C_2$を考える。
\begin{description}
\item[(i)]円$C_1$，$C_2$は2つの不等式$x\geqq0$，$y\geqq0$で定まる領域に含まれる。
\item[(ii)]円$C_1$，$C_2$は直線$l$と同一点で接する。
\item[(iii)]点$C_1$は$x$軸と点$(1, \, 0)$で接し，円$C_2$は$y$軸と接する。
\end{description}
　円$C_1$の半径を$r_1$，円$C_2$の半径を$r_2$とする。
$8r_1+9r_2$が最小となるような直線$l$の方程式と，その最小値を求めよ。\\
