{\huge 5}
　曲線$y=ax^3+bx^2+cx+d$ $(a\neq0)$と$y$軸との交点を$A$とする．
この曲線上の点$P(x,y)$における曲線の接線と$y$軸との交点を$Q$とするとき，
$\displaystyle\lim_{x\to0}\frac{{AQ}^2}{{AP}^2}$，$\displaystyle\lim_{x\to+\infty}\frac{{AQ}^2}{{AP}^2}$を求めよ．
\end{flushleft}
