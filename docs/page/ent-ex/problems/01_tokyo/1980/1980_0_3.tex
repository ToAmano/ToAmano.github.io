　$n$，$a$，$b$，$c$，$d$は0または正の整数であって，
\[ \begin{cases} a^2+b^2+c^2+d^2=n^2-6 \\ a+b+c+d \leqq n \\ a \geqq b \geqq c \geqq d \end{cases} \]
をみたすものとする．このような数の組$(n,a,b,c,d)$をすべて求めよ．
