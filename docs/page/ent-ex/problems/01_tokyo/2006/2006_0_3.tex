{\huge 3}
　$n$を正の整数とする．実数$x$，$y$，$z$に対する方程式
\[ x^n+y^n+z^n=xyz \hspace{1zw} \cdots\cdots\text{\ding{"AC}} \]
を考える．
\begin{description}
\item[(1)]$n=1$のとき，\ding{"AC}を満たす正の整数の組$(x,y,z)$で，$x \leqq y \leqq z$となるものをすべて求めよ．
\item[(2)]$n=3$のとき，\ding{"AC}を満たす正の実数の組$(x,y,z)$は存在しないことを示せ．
\end{description}
\end{flushleft}
