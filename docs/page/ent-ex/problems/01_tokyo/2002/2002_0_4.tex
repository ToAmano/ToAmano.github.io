{\huge 4}
　円周上に$m$個の赤い点と$n$個の青い点を任意の順序に並べる．
これらの点により，円周は$m+n$個の弧に分けられる．
このとき，これらの弧のうち両端の点の色が異なるものの数は偶数であることを証明せよ．
ただし，$m\geqq1$，$n\geqq1$であるとする．
\end{flushleft}
