　2つの放物線
\[ y=2\sqrt3{(x-\cos\theta)}^2+\sin\theta \]
\[ y=-2\sqrt3{(x+\cos\theta)}^2-\sin\theta \]
が相異なる2点で交わるような$\theta$の範囲を求めよ．
ただし，$0^\circ \leqq \theta < {360}^\circ$とする．
