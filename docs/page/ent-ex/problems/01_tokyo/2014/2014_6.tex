{\huge 6}
　座標平面の原点を$O$で表す。\\
　線分$y=\sqrt3 \, x$ $(0 \leqq x \leqq 2)$上の点$P$と，
線分$y=-\sqrt3 \, x$ $(-2 \leqq x \leqq 0)$上の点$Q$が，
線分$OP$と線分$OQ$の長さの和が6となるように動く。
このとき，線分$PQ$の通過する領域を$D$とする。
\begin{description}
\item[(1)]$s$を$0 \leqq s \leqq 2$をみたす実数とするとき，
点$(s, \, t)$が$D$に入るような$t$の範囲を求めよ。
\item[(2)]$D$を図示せよ。
\end{description}
\end{flushleft}
