{\huge 3}
　$u$を実数とする。座標平面上の2つの放物線
\[ C_1:y=-x^2+1 \]
\[ C_2:y={(x-u)}^2+u \]
を考える。
$C_1$と$C_2$が共有点をもつような$u$の値の範囲は，
ある実数$a$，$b$により，$a \leqq u \leqq b$と表される。
\begin{description}
\item[(1)]$a$，$b$の値を求めよ。
\item[(2)]$u$が$a \leqq u \leqq b$をみたすとき，
$C_1$と$C_2$の共有点を$P_1(x_1, \, y_1)$，$P_2(x_2, \, y_2)$とする。
ただし，共有点が1点のみのときは，$P_1$と$P_2$は一致し，
ともにその共有点を表すとする。
\[ 2|x_1y_2-x_2y_1| \]
を$u$の式で表せ。
\item[(3)](2)で得られる$u$の式を$f(u)$とする。定積分
\[ I=\int_a^bf(u)du \]
を求めよ。
\end{description}
\end{flushleft}
