　以下の問いに答えよ。
\begin{description}
\item[(1)]$t$を実数の定数とする。実数全体を定義域とする関数$f(x)$を
\[ f(x)=-2x^2+8tx-12x+t^3-17t^2+39t-18 \]
と定める。このとき，関数$f(x)$の最大値を$t$を用いて表せ。
\item[(2)](1)の「関数$f(x)$の最大値」を$g(t)$とする。
$t$が$\displaystyle t\geqq-\frac{1}{\sqrt2}$の範囲を動くとき，
$g(t)$の最小値を求めよ。
\end{description}
