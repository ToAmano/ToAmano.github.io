{\huge 5}
　平面上の点$(a,b)$の，
直線$2x-y=p$に関する対称点をとり，
次にこれを原点を中心として正の向きに${90}^\circ$回転し，
さらに直線$x+2y=q$に関する対称点をとると，
はじめの点$(a,b)$に一致する．
このとき$p$，$q$を用いて，$a$，$b$を表わせ．
\end{flushleft}
