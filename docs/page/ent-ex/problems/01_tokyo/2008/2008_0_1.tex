　$0 \leqq\alpha\leqq\beta$をみたす実数$\alpha$，$\beta$と，
2次式$f(x)=x^2-(\alpha+\beta)x+\alpha\beta$について，
$\displaystyle\int_{-1}^1f(x)dx=1$が成立しているとする．
このとき定積分$\displaystyle S=\int_0^\alpha f(x)dx$を$\alpha$の式で表し，
$S$がとりうる値の最大値を求めよ．
