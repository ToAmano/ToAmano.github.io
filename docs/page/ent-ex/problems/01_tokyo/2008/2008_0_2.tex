{\huge 2}
　白黒2種類のカードがたくさんある．
そのうち4枚を手もとにもっているとき，次の操作(A)を考える．
\begin{description}
\item[(A)]手持ちの4枚の中から1枚を，等確率$\displaystyle\frac{1}{4}$で選び出し，それを違う色のカードにとりかえる．
\end{description}
最初にもっている4枚のカードは，白黒それぞれ2枚であったとする．以下の(1)，(2)に答えよ．
\begin{description}
\item[(1)]操作(A)を4回繰り返した後に初めて，4枚とも同じ色のカードになる確率を求めよ．
\item[(2)]操作(A)を$n$回繰り返した後に初めて，4枚とも同じ色のカードになる確率を求めよ．
\end{description}
\end{flushleft}
