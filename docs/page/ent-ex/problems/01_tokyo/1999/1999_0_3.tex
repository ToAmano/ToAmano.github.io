　$c$を$\displaystyle c>\frac{1}{4}$を満たす実数とする．
$xy$平面上の放物線$y=x^2$を$A$とし，直線$y=x-c$に関して$A$と対称な放物線を$B$とする．
点$P$が放物線$A$上を動き，点$Q$が放物線$B$上を動くとき，線分$PQ$の長さの最小値を$c$を用いて表せ．
