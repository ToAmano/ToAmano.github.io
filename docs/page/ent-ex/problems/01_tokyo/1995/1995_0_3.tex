{\huge 3}
　$xy$平面において，曲線$y=-x^3+ax$上の$x>0$の部分に，点$P$を次の条件を満たすようにとる．
ただし，$a>0$とする．
\begin{description}
\item[]点$P$におけるこの曲線の接線と$y$軸との交点を$Q$とするとき，
原点$O$における接線が$\angle QOP$を二等分する．
\end{description}
　このとき，$\triangle QOP$の面積$S(a)$の最小値と，それを与える$a$の値を求めよ．
\end{flushleft}
