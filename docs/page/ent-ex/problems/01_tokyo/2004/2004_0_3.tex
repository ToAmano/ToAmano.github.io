　関数$f(x)$，$g(x)$，$h(x)$を次で定める．
\[ f(x)=x^3-3x, \hspace{1zw} g(x)={ \{ f(x) \} }^3-3f(x), \hspace{1zw} h(x)={ \{ g(x) \} }^3-3g(x) \]
このとき，以下の問いに答えよ．
\begin{description}
\item[(1)]$a$を実数とする．$f(x)=a$を満たす実数$x$の個数を求めよ．
\item[(2)]$g(x)=0$を満たす実数$x$の個数を求めよ．
\item[(3)]$h(x)=0$を満たす実数$x$の個数を求めよ．
\end{description}
