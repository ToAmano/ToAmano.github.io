{\huge 2}
　虚部が正の複素数の全体を$H$とする．
すなわち，$H=\{z=x+iy|x,y\text{は実数で}y>0\}$とする．
以下$z$を$H$に属する複素数とする．
$q$を正の実数とし，$\displaystyle f(z)=\frac{z+1-q}{z+1}$とおく．
\begin{description}
\item[(1)]$f(z)$もまた$H$に属することを示せ．
\item[(2)]$f_1(z)=f(z)$と書き，以下$n=2,3,4,\cdots$に対して
\[ f_2(z)=f(f_1(z)), \hspace{1zw} f_3(z)=f(f_2(z)), \hspace{1zw} \cdots, \hspace{1zw}
f_n(z)=f(f_{n-1}(z)), \hspace{1zw} \cdots \]
とおく．
このとき，各$n$に対して
\[ f_n(z)=\frac{r_nz+(1-q)s_n}{s_nz+r_n}, \hspace{1zw} {r_n}^2-(1-q){s_n}^2=q^n \]
が成り立つような$z$によらない実数$r_n$，$s_n$がとれることを示せ．
\end{description}
\end{flushleft}
