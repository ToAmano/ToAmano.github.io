　点$O$を原点とする座標空間内で，一辺の長さが1の正三角形$OPQ$を動かす。
また，点$A(1, \, 0, \, 0)$に対して，$\angle AOP$を$\theta$とおく。
ただし$0^\circ\leqq\theta\leqq{180}^\circ$とする。
\begin{description}
\item[(1)]点$Q$が$(0, \, 0, \, 1)$にあるとき，
点$P$の$x$座標がとりうる値の範囲と，
$\theta$がとりうる値の範囲を求めよ。
\item[(2)]点$Q$が平面$x=0$上を動くとき，辺$OP$が通過しうる範囲を$K$とする。
$K$の体積を求めよ。
\end{description}
