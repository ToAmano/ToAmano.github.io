{\huge 4}
　放物線$y=x^2$のうち$-1 \leqq x \leqq 1$をみたす部分を$C$とする。
座標平面上の原点$O$と点$A(1, \, 0)$を考える。
\begin{description}
\item[(1)]点$P$が$C$上を動くとき，
\[ \overrightarrow{OQ}=2\overrightarrow{OP} \]
をみたす点$Q$の軌跡を求めよ。
\item[(2)]点$P$が$C$上を動き，点$R$が線分$OA$上を動くとき，
\[ \overrightarrow{OS}=2\overrightarrow{OP}+\overrightarrow{OR} \]
をみたす点$S$が動く領域を座標平面上に図示し，その面積を求めよ。
\end{description}
\end{flushleft}
