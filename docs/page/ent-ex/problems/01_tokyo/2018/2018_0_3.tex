　$a>0$とし，
\[ f(x)=x^3-3a^2x \]
とおく。
\begin{description}
\item[(1)]$x\geqq1$で$f(x)$が単調に増加するための，$a$についての条件を求めよ。
\item[(2)]次の2条件をみたす点$(a, \, b)$の動きうる範囲を求め，座標平面上に図示せよ。\\
条件１：方程式$f(x)=b$は相異なる3実数解をもつ。\\
条件２：さらに，方程式$f(x)=b$の解を
$\alpha<\beta<\gamma$とすると$\beta>1$である。
\end{description}
