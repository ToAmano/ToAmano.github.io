　$xyz$空間内の点$P(0,0,1)$を中心とする半径1の球面$K$がある．\\
　$K$上の点$Q(a,b,c)$が条件$a>0$，$b>0$，$c>1$のもとで$K$上を動くとき，
$Q$において$K$に接する平面を$L$とし，
$L$が$x$軸，$y$軸，$z$軸と交わる点をそれぞれ$A$，$B$，$C$とする．
このような三角形$ABC$の面積の最小値を求めよ．
