{\huge 2}
　点$(x,y)$を点$(x+a,y+b)$にうつす平行移動によって
曲線$y=x^2$を移動して得られる曲線を$C$とする．
$C$と曲線$\displaystyle y=\frac{1}{x}$，$x>0$が接するような$a$，$b$を座標とする
点$(a,b)$の存在する範囲の概形を図示せよ．\\
　また，この二曲線が接する点以外に共有点を持たないような$a$，$b$の値を求めよ．
ただし，二曲線がある点で接するとは，その点で共通の接線を持つことである．
\end{flushleft}
