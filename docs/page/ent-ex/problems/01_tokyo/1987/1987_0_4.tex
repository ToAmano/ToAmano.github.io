{\huge 4}
　三つの実数$x$，$y$，$z$のうち最大の数を$max(x,y,z)$で表し，最小の数を$min(x,y,z)$で表す．
いま，次の条件をみたす$x$，$y$，$z$を座標とする点全体の集合を$R$とする．
\[ x\geqq0, \hspace{1zw} y\geqq0, \hspace{1zw} z\geqq0 \]
\[ max(x,y,z) \leqq a \]
\[ x+y+z-min(x,y,z) \leqq a+b \]
$R$の体積を求めよ．
ただし，$a$，$b$は定数で，$a>b>0$とする．
\end{flushleft}
