{\huge 3}
　ある硬貨を投げるとき，表と裏がおのおの確率$\displaystyle\frac{1}{2}$で出るものとする．\\
　この硬貨を8回くり返して投げ，
$n$回目に表が出れば$X_n=1$，
裏が出れば$X_n=-1$とし，
$S_n=X_1+X_2+\cdots+X_n$ $(1 \leqq n \leqq 8)$とおく．\\
　このとき次の確率を求めよ．
\begin{description}
\item[(1)]$S_n\neq0$かつ$S_8=2$となる確率．
\item[(2)]$S_4=0$かつ$S_8=2$となる確率．
\end{description}
\end{flushleft}
