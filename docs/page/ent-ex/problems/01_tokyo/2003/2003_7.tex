\item[(1)]$x\geqq0$のとき，次の不等式を示せ．
\[ x-\frac{x^3}{3 \, !} \leqq \sin x\leqq x-\frac{x^3}{3 \, !}+\frac{x^5}{5 \, !} \]
\item[(2)]曲線$y=\sin x$ $(0 \leqq x \leqq \pi)$と$x$軸で囲まれた図形を
$x$軸のまわりに1回転してできる立体を考える．
この立体を$x$軸に垂直な$2n-1$個の平面によって体積が等しい$2n$個の部分に分割する．
ただし$n$は2以上の自然数である．
\begin{description}
\item[(a)]これら$2n-1$個の平面と$x$軸との交点の$x$座標のうち，
$\displaystyle\frac{\pi}{2}$より小さくかつ$\displaystyle\frac{\pi}{2}$に最も近いものを$a_n$とする．
このとき$\displaystyle\lim_{n\to\infty} n \left( \frac{\pi}{2}-a_n \right)$を求めよ．
\item[(b)]$2n-1$個の平面と$x$軸との交点の$x$座標のうち最も小さいものを$b_n$とする．
数列$\{ n^pb_n \}$が$n\to\infty$のとき0でない有限な値に収束するような実数$p$の値を求めよ．
また，$p$をそのようにとったとき$\displaystyle\lim_{n\to\infty}n^pb_n$を求めよ．
\end{description}
\end{description}
