{\huge 3}
　$C$を放物線$\displaystyle y=\frac{3}{2}x^2-\frac{1}{3}$とする．
$C$上の点$\displaystyle Q \left( t,\frac{3}{2}t^2-\frac{1}{3} \right)$を通り，
$Q$における$C$の接線と垂直な直線を，
$Q$における$C$の法線という．
\begin{description}
\item[(1)]$xy$平面上の点$P(x,y)$で$P$を通る$C$の法線が一本だけ引けるようなものの存在範囲を求め，
$xy$平面上に図示せよ．
\item[(2)](1)で求めた範囲と
放物線の内部(不等式$\displaystyle y>\frac{3}{2}x^2-\frac{1}{3}$の定める範囲)の
共通部分の面積を求めよ．
\end{description}
