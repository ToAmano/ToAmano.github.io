{\huge 3}
　$n$を2以上の整数とする．
$x^n+ax+b$ $(a\text{，}b\text{は実数の定数})$の形の多項式$f(x)$で
\[ \int_{-1}^0f(x)dx=0, \hspace{1zw} \int_{-1}^1f(x)dx=0 \]
を満たすものを求めよ．
この$f(x)$に対して$\displaystyle F(x)=\int_{-1}^xf(t)dt$，$\displaystyle G(x)=\int_{-1}^xF(t)dt$とおく．
$G(x)$が極大または極小となる点$x$と，その点における$G(x)$の値を求めよ．
\end{flushleft}
