{\huge 3}
　$xyz$空間内の原点を中心とする半径1の球面$S=\{(x,y,z)|x^2+y^2+z^2=1,x,y,z\text{は実数}\}$を考え，
$S$上の定点$(0,0,1)$を$A$とする．
$A$とことなる$S$上の点$P(x,y,z)$に対し，直線$AP$と$xy$平面の交点を$Q(u,v,0)$とする．\\
　$k$を正の定数とし，点$P$が$x^2+y^2+z^2=1$，
$\displaystyle x\geqq\frac{1}{k}$，$\displaystyle y\geqq\frac{1}{k}$，$\displaystyle z\geqq\frac{1}{k}$
を満たしながら動くとき，
対応する点$Q$の動く範囲を$uv$平面上に図示せよ．
\end{flushleft}
