{\huge 5}
　$p$，$q$を2つの正の整数とする．
整数$a$，$b$，$c$で条件
\[ -q \leqq b \leqq 0 \leqq a \leqq p, \hspace{1zw} b \leqq c \leqq a \]
を満たすものを考え，
このような$a$，$b$，$c$を$[a, \, b \, ; \, c]$の形に並べたものを$(p, \, q)$パターンと呼ぶ．
各$(p, \, q)$パターン$[a \, ,b \, ; \, c]$に対して
\[ w([a, \, b \, ; \, c])=p-q-(a+b) \]
とおく．
\begin{description}
\item[(1)]$(p, \, q)$パターンのうち，$w([a, \, b \, ; \, c])=-q$となるものの個数を求めよ．
また，$w([a, \, b \, ; \, c])=p$となる$(p, \, q)$パターンの個数を求めよ．
\end{description}
　以下$p=q$の場合を考える．
\begin{description}
\item[(2)]$s$を整数とする．
$(p, \, p)$パターンで$w([a, \, b \, ; \, c])=-p+s$となるものの個数を求めよ．
\item[(3)]$(p, \, p)$パターンの総数を求めよ．
\end{description}
\end{flushleft}
