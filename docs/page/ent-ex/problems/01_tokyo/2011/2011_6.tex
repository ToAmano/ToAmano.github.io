\item[(2)]次の条件を満たす点$(x,y)$全体からなる座標平面内の領域を$S$とする．
\begin{description}
\item[]$x>0$かつ，実数$z$で$0 \leqq t \leqq 1$の範囲の全ての実数$t$に対して
\[ 0 \leqq xt^2 + yt + z \leqq 1 \]
を満たすようなものが存在する．
\end{description}
$S$の概形を図示せよ．
\item[(3)]次の条件を満たす点$(x,y,z)$全体からなる座標空間内の領域を$V$とする．
\begin{description}
\item[]$0 \leqq x \leqq 1$かつ，$0 \leqq t \leqq 1$の範囲の全ての実数$t$に対して，
\[ 0 \leqq xt^2 + yt + z \leqq 1 \]
が成り立つ．
\end{description}
$V$の体積を求めよ．
\end{description}
