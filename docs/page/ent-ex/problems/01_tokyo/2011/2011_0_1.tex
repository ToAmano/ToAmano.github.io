{\huge 1}
　$x$の3次関数$f(x)=ax^3+bx^2+cx+d$が，3つの条件
\[ f(1)=1, \hspace{1zw} f(-1)=-1, \hspace{1zw} \int_{-1}^1(bx^2+cx+d)dx=1 \]
を全て満たしているとする．このような$f(x)$の中で定積分
\[ I=\int_{-1}^\frac{1}{2} {\{ f''(x) \}}^2 dx \]
を最小にするものを求め，そのときの$I$の値を求めよ．
ただし，$f''(x)$は$f'(x)$の導関数を表す．
\end{flushleft}
