{\huge 2}
　座標平面上に8本の直線
\[ x=a \hspace{1zw} (a=1, \, 2, \, 3, \, 4), \hspace{1zw}
y=b \hspace{1zw} (b=1, \, 2, \, 3, \, 4) \]
がある。以下，16個の点
\[ (a, \, b) \hspace{1zw} (a=1, \, 2, \, 3, \, 4, \hspace{1zw} b=1, \, 2, \, 3, \, 4) \]
から異なる5個の点を選ぶことを考える。
\begin{description}
\item[(1)]次の条件を満たす5個の点の選び方は何通りあるか。\\
上の8本の直線のうち，選んだ点を1個も含まないものがちょうど2本ある。
\item[(2)]次の条件を満たす5個の点の選び方は何通りあるか。\\
上の8本の直線は，いずれも選んだ点を少なくとも1個含む。
\end{description}
\end{flushleft}
