{\huge 3}
　空間に2直線$l$，$g$がある．
$l$，$g$の上にそれぞれ3点$A_1$，$A_2$，$A_3$;$B_1$，$B_2$，$B_3$がこの順にあって，
$A_1A_2=B_1B_2$，$A_2A_3=B_2B_3$であるとする．
線分$A_1B_1$，$A_2B_2$，$A_3B_3$の中点をそれぞれ$M_1$，$M_2$，$M_3$とするとき，
3点$M_1$，$M_2$，$M_3$は同一直線の上にあることを証明せよ．
