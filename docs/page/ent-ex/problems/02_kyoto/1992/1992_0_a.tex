　$k$は0または正の整数とする．
方程式$x^2-y^2=k$の解$(a,b)$で，$a$，$b$がともに奇数であるものを奇数解とよぶ．
\begin{description}
\item[(1)]方程式$x^2-y^2=k$が奇数解をもてば，$k$は8の倍数であることを示せ．
\item[(2)]方程式$x^2-y^2=k$が奇数解をもつための必要十分条件を求めよ．
\end{description}
\end{flushleft}
