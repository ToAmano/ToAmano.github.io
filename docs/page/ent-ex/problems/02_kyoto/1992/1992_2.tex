{\huge 2}
　$\theta$は$\displaystyle 0<\theta<\frac{\pi}{2}$の範囲の角とする．
\begin{description}
\item[(1)]$\sin3\theta=\sin2\theta$を満たす$\theta$を求めよ．
\item[(2)]$m$，$n$を0以上の整数とする．
$\theta$についての方程式$\sin3\theta=m\sin2\theta+n\sin\theta$が解を持つときの$(m,n)$と，
そのときの解$\theta$を求めよ．
\end{description}
\end{flushleft}
