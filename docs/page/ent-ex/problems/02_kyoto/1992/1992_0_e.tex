\begin{description}
\item[(1)]$n$を自然数とする(ただし$n\geqq2$)．
0と1からなる数列$x_1,x_2,\cdots\cdots,x_n$で，同じ数が3個以上は続いて並ばないものを考える．
このような数列のうち，$x_{n-1}=x_n$を満たすものの個数を$a_n$，$x_{n-1}\neq x_n$を満たすものの個数を$b_n$とおく．
$a_{n+1}$と$b_{n+1}$は，それぞれ$a_n$，$b_n$によってどのように表わされるか．
\item[(2)]硬貨をくり返し投げる．
3回続けて同じ面が出たら，そこで投げるのをやめる．
ちょうど$n$回投げてやめる確率を$p_n$とおく．
$p_7$を求めよ．
\end{description}
