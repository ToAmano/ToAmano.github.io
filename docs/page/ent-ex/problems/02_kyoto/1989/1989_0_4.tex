{\huge 4}
　関数$y=x^3-ax$のグラフ上の点$P$における接線$T_P$がこのグラフと再び交わる点を$Q$とする．
ただし，$T_P$がこのグラフと共有する点が$P$以外にないときは$Q=P$と定める．
\begin{description}
\item[(1)]$P$の$x$座標を$c$として，$Q$の座標を求めよ．
\item[(2)]点$Q$におけるこのグラフへの接線が$T_P$と直交するような$P$は何個あるか．
\end{description}
\end{flushleft}
