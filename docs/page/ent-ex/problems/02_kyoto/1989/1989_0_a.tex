　縦40cm，横25cmの長方形の紙がある．
その四隅から，一辺の長さ$x$cmの正方形を切り取り，残りの紙を折りまげて，直方体の形の，ふたのない容器を作る．
このとき，この箱の容積を$V\text{cm}^3$とする．
$V$を$x$の式で表せ．
また$V$が最大となる$x$の値を求めよ．
