{\huge 6}
　$\displaystyle F(x)=\frac{ax}{x+1}$とおく．
ただし，$a$は定数で$0<a\leqq1$である．
関数の列$\{f_n(x)\}$を次によって定める．
\begin{description}
\item[(i)]$f_1(x)=F(x)$
\item[(ii)]$f_{n+1}(x)=F(f_n(x))$ $(n=1,2,3,\cdots)$
\end{description}
\begin{description}
\item[(1)]$f_n(x)$を$a$，$x$，$n$の式で表せ．
\item[(2)]次の条件をみたす数列$\{b_n\}$を一つ作れ．($a$の値によって，異なる数列であってもよい．)\\
$[$条件$]$$c>0$ならば，数列$\{b_n\cdot f_n(c)\}$は正の数に収束する．
\end{description}
\end{flushleft}
