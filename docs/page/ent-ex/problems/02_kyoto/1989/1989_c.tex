　座標平面において，
つぎの条件をみたす$\triangle ABC$と半平面$H=\{(x,y)|x\geqq0\}$との共通部分の面積の最大値を求めよ．\\
　$\triangle ABC$は$AB=AC$であるような二等辺三角形であって，$BC$は$y$軸に平行で，$A$の座標は$(-1,0)$である．
また，$AB$と$y$軸との交点を$D$とすると，$DB=2\sqrt3$である．
