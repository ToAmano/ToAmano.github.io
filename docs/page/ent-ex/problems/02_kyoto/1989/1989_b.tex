{\huge 2}
　放物線$y=x^2$を$y$軸のまわりに回転してできる曲面があり，
$y$軸が水平面に垂直で$y$軸の正の部分が上方にあるように置いてある．
その曲面の中に半径$r$ $\displaystyle \left( r>\frac{1}{2} \right)$の球を落とし込む．\\
　このとき，この回転面と球面とで囲まれた部分の体積を求めよ．
