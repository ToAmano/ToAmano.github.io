　$x$，$y$を相異なる正の実数とする．
数列$\{ a_n \}$を$a_1=0$，$a_{n+1}=xa_n+y^{n+1}$ $(n=1,2,3,\cdots)$によって定めるとき，
$\displaystyle\lim_{n\to\infty}a_n$が有限の値に収束するような座標平面上の点$(x,y)$の範囲を図示せよ．
