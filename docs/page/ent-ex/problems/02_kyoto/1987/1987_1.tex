{\huge 1}
　$x$の3次式
\begin{description}
\item[]$f(x)=ax^3+(a^2+b)x^2+(2ab+c)x+a^2+b^2-a$
\item[]$g(x)=ax^3+(a^2-b)x^2+(a-1)x+c^2-b^2$
\end{description}
および$x$の2次式
\begin{description}
\item[]$h(x)=x^2+ax+b$
\end{description}
を考える．($a$，$b$，$c$は定数，$a\neq0$)\\
　$f(x)$，$g(x)$はともに$h(x)$で割り切れるか，または，ともに$h(x)$では割り切れないかの，いずれかであることを示せ．
\end{flushleft}
