{\huge 5}
　空間において，平面$ax+y+z+a-2=0$($a$は定数)を考える．
\begin{description}
\item[(1)]この平面上の点のうち原点に一番近いものの座標を求めよ．
\item[(2)]原点を中心とする半径3の球体がこの平面で分けられる二つの部分のうち，体積の大きくない方の体積を$V(a)$とする．
$a$をいろいろ変えたとき，$V(a)$が最小になる$a$と，そのときの$V(a)$の値を求めよ．
\end{description}
\end{flushleft}
