{\huge 4}
　4人でジャンケンをし，負けたものが順次去って，残りでジャンケンをして，勝者が一人になるまで続けるものとする．
また，4人はそれぞれ独立に，グー(石)，チョキ(はさみ)，パー(紙)を確率$\displaystyle\frac{1}{3}$で出すものとする．
\begin{description}
\item[(1)]1回でゲームが終了する確率$p_1$および2回目の勝負が$m$人($m=2,3,4$)で行われる確率$p_m$を求めよ．
\item[(2)]2回でこのゲームが終了する確率$q$を求めよ．
\end{description}
