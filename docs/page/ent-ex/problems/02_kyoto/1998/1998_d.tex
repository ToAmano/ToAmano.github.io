{\huge 4}
　$a$は$0<a<\pi$を満たす定数とする．
$n=0,1,2,\cdots$に対し，$n\pi<x<(n+1)\pi$の範囲に$\sin(x+a)=x\sin x$を満たす$x$がただ一つ存在するので，
この$x$の値を$x_n$とする．
\begin{description}
\item[(1)]極限値$\displaystyle\lim_{n\to\infty}(x_n-n\pi)$を求めよ．
\item[(2)]極限値$\displaystyle\lim_{n\to\infty}n(x_n-n\pi)$を求めよ．
\end{description}
\end{flushleft}
