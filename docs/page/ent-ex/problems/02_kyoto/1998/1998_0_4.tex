{\huge 4}
　$xy$平面上で放物線$y=x^2$上に2点$A(a,a^2)$，$B(b,b^2)$ $(a<b)$をとり，
線分$AB$と放物線で囲まれた図形の面積を$s$とする．
点$P(t,t^2)$を放物線上にとり，三角形$ABP$の面積を$S(P)$とする．
$t$が$a<t<b$の範囲を動くときの$S(P)$の最大値を$S$とするとき，$s$と$S$の比を求めよ．
\end{flushleft}
