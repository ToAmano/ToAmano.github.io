\item[(1)]$m$，$n$を2つの正整数とする．
$\cos m^\circ$，$\sin m^\circ$，$\cos n^\circ$，$\sin n^\circ$のすべてが有理数であるとき，
$\cos{(m+n)}^\circ$，$\sin{(m+n)}^\circ$はともに有理数であることを示せ．
\item[(2)]$n$は60の約数とする．
このとき，$\cos n^\circ$と$\sin n^\circ$のうち，少なくとも一方は無理数であることを示せ．
\end{description}
