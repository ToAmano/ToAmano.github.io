{\huge 3}
　点$O$を中心とする半径1の球面上に4点$A$，$B$，$C$，$D$があって
\[ \overrightarrow{OA}+\overrightarrow{OB}+\overrightarrow{OC}+\overrightarrow{OD}=\vec{0} \]
が成立しているとする．
\begin{description}
\item[(1)]$|\overrightarrow{AB}|=|\overrightarrow{CD}|$であることを示せ．
\item[(2)]点$B'$，$D'$を$\overrightarrow{OB'}=-\overrightarrow{OB}$，
$\overrightarrow{OD'}=-\overrightarrow{OD}$となるようにとる．
このとき，$A$，$B'$，$C$，$D'$が互いに異なるならば，これら4点は，この順で，ある長方形の頂点となっていることを示せ．
\end{description}
\end{flushleft}
