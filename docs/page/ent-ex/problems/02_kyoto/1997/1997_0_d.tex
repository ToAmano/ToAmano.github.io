　3角形$\triangle ABC$において，$\alpha=\sin2A$，$\beta=\sin2B$，$\gamma=\sin2C$とおくとき，
次の2つの条件(イ)，(ロ)は互いに同値((イ)$\Longleftrightarrow$(ロ))であることを示せ．
\begin{description}
\item[(1)]$\alpha^2=\beta^2+\gamma^2$．
\item[(2)]$A={45}^\circ$，または$A={135}^\circ$，または$B={90}^\circ$，または$C={90}^\circ$．
\end{description}
