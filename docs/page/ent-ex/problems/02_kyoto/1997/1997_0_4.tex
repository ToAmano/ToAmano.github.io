　2次関数$y={(ax+b)}^2$ $(0 \leqq x \leqq 1)$の最大値を$M(a,b)$とする．
このとき，次の不等式(＊)が任意の実数$a$，$b$に対して成り立つような実数$m$の中で最小のものを求めよ．
\begin{description}
\item[(＊)]$\displaystyle M(a,b) \leqq m\int_0^1{(ax+b)}^2dx$
\end{description}
