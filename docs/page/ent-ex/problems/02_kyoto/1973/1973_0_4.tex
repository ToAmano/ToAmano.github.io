{\huge 4}
　$\mathbf{a}$，$\mathbf{b}$，$\mathbf{c}$は平面上の単位ベクトルで，どの二つも${120}^\circ$の角をなすものとする．\\
　このとき，この平面上の任意のベクトル$\mathbf{x}$に対して
\begin{description}
\item[(i)]$(\mathbf{a},\mathbf{x})+(\mathbf{b},\mathbf{x})+(\mathbf{c},\mathbf{x})=0$が成り立つことを示せ．
\item[(ii)]${(\mathbf{a},\mathbf{x})}^2+{(\mathbf{b},\mathbf{x})}^2+{(\mathbf{c},\mathbf{x})}^2$の値を
$\mathbf{x}$の大きさ$l$を用いて表わせ．
\end{description}
　ただし$(\mathbf{a},\mathbf{x})$などはベクトルの内積を表わす．
\end{flushleft}
