{\huge 2}
　定数$c$ $(c\neq0)$に対して，
等式$f(x+c)=f(x)$がすべての$x$について成り立つとき，
関数$f(x)$は周期関数であるといい，
またこの等式を満たすような正の数$c$のうちの最小値を，
$f(x)$の周期という．
次の関数は周期関数であることを示し，
またその周期を求めよ．
\begin{description}
\item[(1)]$f(x)=2^{\sin x}$
\item[(2)]$f(x)=\sin(\sin x)$
\end{description}
\end{flushleft}
