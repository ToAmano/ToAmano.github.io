　次の2つの条件を同時に満たす四角形のうち面積が最小のものの面積を求めよ．
\begin{description}
\item[(a)]少なくとも2つの内角は${90}^\circ$である．
\item[(b)]半径1の円が内接する．
ただし，円が四角形に内接するとは，円が四角形の4つの辺すべてに接することをいう．
\end{description}
