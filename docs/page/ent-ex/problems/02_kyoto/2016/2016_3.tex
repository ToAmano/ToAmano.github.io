{\huge 3}
　四面体OABCが次の条件を満たすならば，それは正四面体であることを示せ．
\begin{description}
\item[条件：]頂点A，B，Cからそれぞれの対面を含む平面へ下ろした垂線は対面の外心を通る．
\end{description}
ただし，四面体のある頂点の対面とは，その頂点を除く他の3つの頂点がなす三角形のことをいう．
