　$xyz$空間において，平面$y=z$の中で
\[ |x|\leqq\frac{e^y+e^{-y}}{2}-1, \hspace{1zw} 0 \leqq y \leqq \log a \]

　この図形$D$を$y$軸のまわりに1回転させてできる立体の体積を求めよ．
