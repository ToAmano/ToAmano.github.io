{\huge 6}
　さいころを$n$回投げて出た目を順に$X_1,X_2,\cdots,X_n$とする．さらに
\[ Y_1=X_1, \hspace{1zw} Y_k=X_k+\frac{1}{Y_{k-1}} \hspace{1zw} (k=2,\cdots,n) \]
によって$Y_1,Y_2,\cdots,Y_n$を定める．
\[ \frac{1+\sqrt3}{2} \leqq Y_n \leqq 1+\sqrt3 \]
となる確率$p_n$を求めよ．
\end{flushleft}
