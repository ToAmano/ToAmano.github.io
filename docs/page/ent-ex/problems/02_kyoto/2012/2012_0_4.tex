{\huge 4}
　次の命題(p),(q)のそれぞれについて，正しいかどうか答えよ．
正しければ証明し，正しくなければ反例を挙げて正しくないことを説明せよ．
\begin{description}
\item[(p)]正$n$角形の頂点から3点を選んで内角の1つが$60^\circ$である
三角形を作ることができるならば，$n$は3の倍数である．
\item[(q)]$\triangle ABC$と$\triangle A'B'C'$において，
$AB=A'B'$，$BC=B'C'$，$\angle A = \angle A'$ならば，
これら2つの三角形は合同である．
\end{description}
\end{flushleft}
