{\huge 2}
　平面上の正方形の列$S_n$ $(n=1,2,\cdots)$を
次の条件\ding{"AC}，\ding{"AD}，\ding{"AE}，\ding{"AF}を満たすようにとる．
\begin{description}
\item[\ding{"AC}]$S_n$の1つの辺は$x$軸に含まれる．
\item[\ding{"AD}]$S_n$の1つの頂点は双曲線$xy=1$上にある．
\item[\ding{"AE}]$S_{n+1}$は$S_n$と異なり，$S_{n+1}$の1つの辺は$S_n$のある辺に含まれる．
\item[\ding{"AF}]$S_1$の頂点は$(0,0)$，$(0,1)$，$(1,0)$，$(1,1)$である．
\end{description}
$S_n$の頂点で$xy=1$上にあるものの座標を$(x_n,y_n)$とする．
このとき${x_n}^2 \geqq n$となることを示せ．
\end{flushleft}
