{\huge 2}
　$\alpha$，$\beta$は複素数で，$\alpha$の絶対値は1とする．
このとき$z+\alpha\bar{z}+\beta=0$を満足する複素数$z$があるための必要十分条件は$\alpha\bar{\beta}=\beta$であることを示せ．
ここに$\bar{z}$，$\bar{\beta}$はそれぞれ$z$，$\beta$の共役複素数を表わす．
\end{flushleft}
