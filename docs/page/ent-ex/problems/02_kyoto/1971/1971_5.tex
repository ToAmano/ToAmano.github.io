{\huge 5}
　半径$r$の円に内接する正$n$角形の頂点を順次$A_1,A_2,\cdots\cdots,A_n$とする．
まず頂点$A_2$を中心とする半径$A_2A_1$の円と辺$A_3A_2$の延長($\overrightarrow{A_3A_2}$の方向)との交点を$B_1$として
扇形$A_2A_1B_1$をつくる．
次に頂点$A_3$を中心とする半径$A_3B_1$の円と辺$A_4A_3$の延長($\overrightarrow{A_4A_3}$の方向)との交点を$B_2$として
扇形$A_3B_1B_2$をつくる．
順次このようにして$n$個の扇形をつくる．
さて，正$n$角形$A_1A_2\cdots\cdots A_n$の面積とこれら$n$個の扇形の面積の総和を$S_n$で表わすとき，
\begin{description}
\item[(i)]$S_n$を$n$，$r$を用いて表わせ．
\item[(ii)]$\displaystyle\lim_{n\to\infty}S_n$を求めよ．
\end{description}
\end{flushleft}
