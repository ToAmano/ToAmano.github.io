　$a$，$b$，$c$は与えられた正の定数で，$a<b<c$であるとする．
$-\infty < x \leqq c$の範囲で，3次函数$y=x(x-a)(b-x)$のグラフが，2次函数$y=px^2-qx$のグラフの下側に出ないように，
正の定数$p$，$q$の値を求めることができるかどうかを吟味せよ．
