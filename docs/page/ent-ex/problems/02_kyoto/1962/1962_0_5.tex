{\huge 5}
　各点$P(x,y)$に対して実数値$z$をさだめようとするのに，2次方程式
\[ z^2+(a+x)z+\frac{1}{4}(x^2-y^2)=0 \hspace{1zw} (a\text{は正の定数}) \]
が異なる2実根をもつ場合には大きい方の根の値を採ることにし，
他の場合には$z$の値を定めないことにする．
こうして，正の値$z$が定まるような点$P$の範囲を明らかにし，この範囲をできるだけ正確に図示せよ．
\end{flushleft}
