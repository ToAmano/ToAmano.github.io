{\huge 4}
　$x$，$y$は$x+y>0$，$x-y>0$をみたす実数とする．
ある4面体の隣り合う2辺の長さが$\sqrt{x+y}$，$\sqrt{x-y}$で，残り4辺の長さはすべて1であるという．
このような条件をみたす点$(x,y)$の存在範囲を図示せよ．
