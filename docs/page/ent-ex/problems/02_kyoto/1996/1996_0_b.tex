{\huge 2}
　$n$は2以上の自然数，$p$は素数，$a_0$，$a_1$，$\cdots$，$a_{n-1}$は整数とし，
$n$次式$f(x)=x^n+pa_{n-1}x^{n-1}+\cdots+pa_ix^i+\cdots+pa_0$を考える．
\begin{description}
\item[(1)]方程式$f(x)=0$が整数解$\alpha$を持てば，$\alpha$は$p$で割り切れることを示せ．
\item[(2)]$a_0$が$p$で割り切れなければ，方程式$f(x)=0$は整数解を持たないことを示せ．
\end{description}
\end{flushleft}
