　三角形$ABC$の辺上を動く点$D$，$E$，$F$が，時刻$t=0$にそれぞれ$A$，$B$，$C$を出発し，
$B$，$C$，$A$にむかってそれぞれ一定の速さで進んで，時刻$t=1$に$B$，$C$，$A$に達するものとする．

\item[(ロ)]$\triangle DEF$の面積の最小値は$\triangle ABC$の面積の何倍か．
\end{description}
