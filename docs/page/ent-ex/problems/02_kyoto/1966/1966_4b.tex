{\huge 4 旧}
　二次方程式$x(x-a)+m(x-b)(x-c)=0$が$m$のすべての正の値に対して実根をもつための必要十分条件を
$a$，$b$，$c$の大小関係によって表わせ．
ただし$a$，$b$，$c$は相異なる正数とする．
\end{flushleft}
