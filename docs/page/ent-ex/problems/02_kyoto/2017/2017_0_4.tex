{\huge 4}
　$p$，$q$を自然数，$\alpha$，$\beta$を
\[ \tan\alpha=\frac{1}{p}, \hspace{1zw} \tan\beta=\frac{1}{q} \]
を満たす実数とする．このとき，次の問に答えよ．
\begin{description}
\item[(1)]次の条件
\[ \text{(A)} \hspace{1zw} \tan(\alpha+2\beta)=2 \]
を満たす$p$，$q$の組$(p, \, q)$のうち，
$q\leqq3$であるものをすべて求めよ．
\item[(2)]条件(A)を満たす$p$，$q$の組$(p, \, q)$で，
$q>3$であるものは存在しないことを示せ．
\end{description}
\end{flushleft}
