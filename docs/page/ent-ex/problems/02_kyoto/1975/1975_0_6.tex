{\huge 6}
　$a$が実数で$a<1$のとき，数列$x_0,x_1,x_2,\cdots,x_n,\cdots$を
$x_0=a$，$\displaystyle x_n=\frac{1}{2-x_{n-1}}$ $(n=1,2,3,\cdots)$によって定義する．
このとき
\begin{description}
\item[(i)]$x_n$を$n$と$a$で表わせ．
\item[(ii)]$\displaystyle\lim_{n\to\infty}n^2 \left( x_n-1+\frac{1}{n} \right)$を求めよ．
\end{description}
\end{flushleft}
