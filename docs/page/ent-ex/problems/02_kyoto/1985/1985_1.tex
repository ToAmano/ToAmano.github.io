　実数$p$，$q$ $(q>0)$に対して，
下の2条件\ding{"AC}，\ding{"AD}を満たす三角形$ABC$が存在するための必要十分条件を求めよ．
\begin{description}
\item[\ding{"AC}]$|\overrightarrow{BC}|=q$
\item[\ding{"AD}]$\overrightarrow{AB}\cdot\overrightarrow{AC}=p$
\end{description}
ただし，$\overrightarrow{AB}\cdot\overrightarrow{AC}$は
$\overrightarrow{AB}$と$\overrightarrow{AC}$の内積をあらわす．
