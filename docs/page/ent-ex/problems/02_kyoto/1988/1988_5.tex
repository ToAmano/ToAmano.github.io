{\huge 5}
　一辺の長さが1の正方形と，一辺の長さが2の正方形でできたタイルが多数大きな袋に入っている．
でたらめにタイルを一枚取り出すとき，
小さいタイルが取り出される確率を$p$，大きいタイルが取り出される確率を$q=1-p$とする．
縦の長さが3，横の長さが6の長方形に，取り出されたタイルを敷きつめていくとき，
ちょうど$n$回でこの長方形が過不足なく敷きつめられる確率を求めよ．
\end{flushleft}
