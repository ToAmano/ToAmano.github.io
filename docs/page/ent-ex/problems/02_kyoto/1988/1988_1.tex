　実数$a$，$b$に対し，$f(x)=x^2+ax+b$，$g(x)=f(f(x))$とする．
\begin{description}
\item[(1)]$g(x)-x$は$f(x)-x$で割り切れることを示せ．
\item[(2)]$g(p)=p$かつ$f(p) \neq p$をみたす実数$p$が存在するような点$(a,b)$の範囲を図示せよ．
\end{description}
\end{flushleft}
