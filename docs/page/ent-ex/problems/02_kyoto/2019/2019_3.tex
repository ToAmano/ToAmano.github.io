{\huge 3}
　鋭角三角形ABCを考え，その面積をSとする。
$0<t<1$をみたす実数$t$に対し，線分ACを$t:1-t$に内分する点をQ，
線分BQを$t:1-t$に内分する点をPとする。
実数$t$がこの範囲を動くときに点Pの描く曲線と，
線分BCによって囲まれる部分の面積を，Sを用いて表せ。
