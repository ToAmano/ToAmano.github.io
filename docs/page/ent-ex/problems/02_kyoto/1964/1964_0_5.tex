{\huge 5}
　$a$，$b$は1でない正の数とする．
$x$の2次方程式$\displaystyle x^2-{(\log_ab)}^2x+\log_a \left( \frac{1}{b^2} \right) =0$がある．
\begin{description}
\item[(イ)]この方程式が$\displaystyle \log_b \left( \frac{1}{a} \right)$より大きい2根(重根でもよい)を
もつような点$(a,b)$の範囲を求め，それを図示せよ．
\item[(ロ)]また，上の2次方程式の2根を$\alpha$，$\beta$として，
$(a,b)$が(イ)の条件の範囲を動くとき，
$12\alpha+12\beta-\alpha^2\beta-\alpha\beta^2$のとりうる値の最大値を求めよ．
\end{description}
\end{flushleft}
