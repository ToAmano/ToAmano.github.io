{\huge 3}
　整数$n$に対し$\displaystyle f(n)=\frac{n(n-1)}{2}$とおき，$a_n=i^{f(n)}$と定める．
ただし，$i$は虚数単位を表す．
このとき，
$a_{n+k}=a_n$が任意の整数$n$に対して成り立つような正の整数$k$をすべて求めよ．
\end{flushleft}
