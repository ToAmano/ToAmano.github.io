{\huge 5}
　$p$を2以上の整数とする．
2以上の整数$n$に対し，
次の条件(イ)，(ロ)をみたす複素数の組$(z_1,z_2,\cdots,z_n)$の個数を$a_n$とする．
\begin{description}
\item[(イ)]$k=1,2,\cdots,n$に対し，${z_k}^p=1$かつ，$z_k\neq1$．
\item[(ロ)]$z_1z_2\cdots z_n=1$．
\end{description}
　このとき，次の問いに答えよ．
\begin{description}
\item[(1)]$a_3$を求めよ．
\item[(2)]$a_{n+2}$を$a_n$，$a_{n+1}$の一方または両方を用いて表せ．
\item[(3)]$a_n$を求めよ．
\end{description}
\end{flushleft}
