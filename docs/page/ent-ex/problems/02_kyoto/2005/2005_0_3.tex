{\huge 3}
　$\alpha$，$\beta$は0でない相異なる複素数で，
$\displaystyle\frac{\alpha}{\beta}+\frac{\bar{\alpha}}{\bar{\beta}}=2$を満たすとする．
このとき，0，$\alpha$，$\beta$の表す複素平面上の3点を結んで得られる三角形はどのような三角形か．
(ただし，複素数$z$に対し，$\bar{z}$は$z$に共役な複素数である．また，複素平面を複素数平面ともいう．)
\end{flushleft}
