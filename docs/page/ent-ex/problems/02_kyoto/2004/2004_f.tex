{\huge 6}
　$n$を自然数とする．
$xy$平面内の，原点を中心とする半径$n$の円の，内部と周をあわせたものを$C_n$であらわす．
次の条件(＊)を満たす1辺の長さが1の正方形の数を$N(n)$とする．
\begin{description}
\item[(＊)]正方形の4頂点はすべて$C_n$に含まれ，4頂点の$x$および$y$座標はすべて整数である．
\end{description}
　このとき，$\displaystyle\lim_{n\to\infty}\frac{N(n)}{n^2}=\pi$を証明せよ．
\end{flushleft}
