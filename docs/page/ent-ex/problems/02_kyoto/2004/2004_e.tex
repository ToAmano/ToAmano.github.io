{\huge 5}
　$n$を自然数とする．
次の3つの不等式(1)，(2)，(3)をすべて満たす自然数の組$(a,b,c,d)$はいくつあるか．
$n$を用いてあらわせ．
\begin{description}
\item[(1)]$1 \leqq a < d \leqq n$
\item[(2)]$a \leqq b < d$
\item[(3)]$a < c \leqq d$
\end{description}
\end{flushleft}
