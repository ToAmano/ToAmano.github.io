　$x\geqq0$に対して，関数$f(x)$を次のように定義する．
\[ f(x)=\begin{cases} x & (0 \leqq x \leqq 1 \text{のとき}) \\ 0 & (x>1\text{のとき}) \end{cases} \]
　自然数$k$に対して，$\displaystyle\int_0^3f \left( \frac{x^2}{k} \right) dx$を求めよ．
