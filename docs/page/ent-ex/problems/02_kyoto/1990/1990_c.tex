{\huge 3}
　関数$y=\log x$のグラフ上の1点$P(s,\log s)$ $(s\geqq1)$における接線と$y$軸の交点を$Q$とする．
グラフ上に定点$A(1,0)$をとる．
$AP$間のグラフの長さを$\stackrel{\frown}{AP}$，線分$PQ$の長さを$\overline{PQ}$とし，
$t=\overline{PQ}-\stackrel{\frown}{AP}$とする．\\
　$t$は$s$の関数である．
\begin{description}
\item[(1)]$\displaystyle\frac{dt}{ds}$を$s$で表わせ．
\item[(2)]$\displaystyle u=\frac{1}{s}$，$v=\sqrt{1+u^2}$とおくとき，
$\displaystyle\frac{du}{dt}$および$\displaystyle\frac{dv}{dt}$を$u$の関数として表わせ．
\item[(3)]$u$を$t$の関数として表わせ．
\end{description}
\end{flushleft}
