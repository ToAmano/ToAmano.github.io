{\huge 6}
　円$C:x^2+y^2=1$を内部に含む楕円$\displaystyle D:\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ $(a>0,b>0)$がある．
$D$上の1点$P(0,b)$から$C$に1つの接線をひき，その延長が再び$D$と交わる点を$Q$とする．
$Q$から$C$に$PQ$とは異なる接線をひき，その延長が再び$D$と交わる点を$R$とする．
$R$から$C$に$QR$と異なる接線をひき，その延長が再び$D$と交わる点を$S$とすると，$S=P$となった．
このとき$a$を$b$の関数として表わせ．
\end{flushleft}
