　$O$を原点とし，$y=x^3-x$のグラフを$C$とする．
$C$上の2点$P$，$P'$は線分$OP$と$OP'$が直交するように$C$上を動き，$P$は第1象限，$P'$は第2象限にある．
$C$と$OP$で囲まれた部分の面積を$S$，$C$と$OP'$で囲まれた部分の面積を$S'$とする．
$\displaystyle\frac{S}{S'}$の最小値を求めよ．
