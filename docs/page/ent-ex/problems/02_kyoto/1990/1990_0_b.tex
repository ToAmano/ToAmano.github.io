　平面上の2定点$A$，$B$に対し，点$C$は線分$AB$を直径とする円周上を動く．
直線$AB$に関して$C$と同じ側に3点$A'$，$B'$，$C'$を次の2条件をみたすようにとる．
\begin{description}
\item[(イ)]$\triangle CBA'$，$\triangle ACB'$，$\triangle ABC'$はいずれも正三角形である．
\item[(ロ)]$\triangle CBA'$，$\triangle ACB'$は$\triangle ABC$と重なりがない．
\end{description}
　このとき，四角形$CA'C'B'$の面積が最大となる$\theta=\angle CAB$の値を求めよ．
\end{flushleft}
