{\huge 4}
　半径1，$1-2r$の同心円の間に半径$r$の円が$n$個，互いに交わらずに入っているという状態を考える．
$n$ $(\geqq2)$を固定した上で，$r$を変化させる．
\begin{description}
\item[(1)]$r$は$\displaystyle0<r\leqq\frac{\displaystyle\sin\frac{\pi}{n}}{\displaystyle1+\sin\frac{\pi}{n}}$
の範囲になければならないことを示せ．
\item[(2)]これら$n+2$個の円の面積の総和が最小となる$r$の値を求めよ．
\end{description}
\end{flushleft}
