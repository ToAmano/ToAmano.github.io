{\huge 2}
　平面上の鋭角三角形$\triangle ABC$の内部(辺や頂点は含まない)に点$P$をとり，
$A'$を$B$，$C$，$P$を通る円の中心，
$B'$を$C$，$A$，$P$を通る円の中心，
$C'$を$A$，$B$，$P$を通る円の中心とする．
このとき$A$，$B$，$C$，$A'$，$B'$，$C'$が同一円周上にあるための必要十分条件は
$P$が$\triangle ABC$の内心に一致することであることを示せ．
\end{flushleft}
