{\huge 6 新}
　3人乗りのボートが2つある．
4人の人をそれらに分乗させるしかたが幾通りあるか．
次のおのおのの場合に求めよ．
\begin{description}
\item[(i)]人もボートも区別しないで，人数の分け方だけを考えるとき，
\item[(ii)]人は区別しないが，ボートは区別するとき，
\item[(iii)]ボートも人も区別を考えるが，どの座席につくかは区別しないとき，
\item[(iv)]ボートも区別し，どの人がどの座席につくかも区別するとき．
\end{description}
\end{flushleft}
