{\huge 5}
　初項$a_1=2$，第2項$a_2=3$であるような数列で，一般項$a_n$が簡単な式で与えられるものを作りたい．
$A$，$B$，$C$，$D$は$n$または定数，$*$は四則演算(＋，−，×，÷)のうちのいずれか，$A^n$は$A$の$n$乗をあらわすとき，
\begin{description}
\item[(i)]$a_n=A*B*C*D$の形の式で与えられる数列
\item[(ii)]$a_n=A^n*B*C$の形の式で与えられる数列
\end{description}
で，$a_1=2$，$a_2=3$をみたすものをそれぞれ3つずつ，合わせて6つの相異なる数列を作れ．
ただし，1つの式の中の$*$は異なる四則演算を使ってもよく，＋0，×1等は省略してよい．
\end{flushleft}
