　数列$\{a_n\}$，$\{b_n\}$を$a_1=3$，$b_1=2$
\[ a_{n+1}={a_n}^2+2{b_n}^2, \hspace{1zw} b_{n+1}=2a_nb_n \hspace{1zw} (n\geqq1) \]
で定める．
\begin{description}
\item[(1)]${a_n}^2-2{b_n}^2$を求めよ．
\item[(2)]$\displaystyle\lim_{n\to\infty}\frac{a_n}{b_n}$を求めよ．
\end{description}
