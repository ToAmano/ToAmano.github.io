　数列$\{ a_n \}$の初項$a_1$から第$n$項$a_n$までの和を$S_n$と表す．
この数列が$a_1=1$，$\displaystyle\lim_{n\to\infty}S_n=1$，$n(n-2)a_{n+2}=S_n$ $(n \geqq 1)$を満たすとき，
一般項$a_n$を求めよ．
