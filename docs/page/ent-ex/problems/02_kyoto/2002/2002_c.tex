　各面が鋭角三角形からなる四面体$ABCD$において，辺$AB$と辺$CD$は垂直ではないとする．
このとき辺$AB$を含む平面$\alpha$に点$C$，点$D$から下ろした垂線の足をそれぞれ$C'$，$D'$とするとき，
4点$A$，$B$，$C'$，$D'$がすべて相異なり，しかも同一円周上にあるように$\alpha$がとれることを示せ．
