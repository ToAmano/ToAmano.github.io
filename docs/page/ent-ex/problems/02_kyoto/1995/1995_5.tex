{\huge 5}
　1番から7番まで番号のついた席が番号順に一列に並んでいる．
客が順に到着して次のように着席していくとする．
\begin{description}
\item[(イ)]両端の席および先客が着席している隣の席に次の客が着席する確率は，すべて等しい．
\item[(ロ)]両端が空席の席に着席する確率は，隣の席にすでに先客が着席している席または端の席に着席する確率に比べて2倍である．
\end{description}
　このとき，
\begin{description}
\item[(1)]3人目の客が到着したときに，すでに1番と3番の席に先客が着席している確率を求めよ．
\item[(2)]4人目の客が到着したときに，すでに2番，4番，6番の席に先客が着席している確率を求めよ．
\end{description}
\end{flushleft}
