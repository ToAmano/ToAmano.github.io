{\huge 5}
　$H>0$，$R>0$とする．
空間内において，原点$O$と点$P(R,0,H)$を結ぶ線分を，$z$軸のまわりに回転させてできる容器がある．
この容器に水を満たし，原点から水面までの高さが$h$のとき単位時間あたりの排水量が，$\sqrt{h}$となるように，水を排出する．
すなわち，時刻$t$までに排出された水の総量を$V(t)$とおくとき，$\displaystyle\frac{dV}{dt}=\sqrt{h}$が成り立つ．
このときすべての水を排出するのに要する時間を求めよ．
\end{flushleft}
