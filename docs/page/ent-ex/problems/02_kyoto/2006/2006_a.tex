　1次式$A(x)$，$B(x)$，$C(x)$に対して
${ \{ A(x) \} }^2+{ \{ B(x) \} }^2={ \{ C(x) \} }^2$が成り立つとする．
このとき$A(x)$と$B(x)$はともに$C(x)$の定数倍であることを示せ．
