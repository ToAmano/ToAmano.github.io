　座標空間において，平面$z=1$上に一辺の長さが1の正三角形$ABC$がある．
点$A$，$B$，$C$から平面$z=0$におろした垂線の足をそれぞれ$D$，$E$，$F$とする．
動点$P$は$A$から$B$の方向へ出発し，一定の速さで$\triangle ABC$の周を一周する．
動点$Q$は同時に$E$から$F$の方向へ出発し，$P$と同じ一定の速さで$\triangle DEF$の周を一周する．
線分$PQ$が通過してできる曲面と$\triangle ABC$，$\triangle DEF$によって囲まれる立体を$V$とする．
\begin{description}
\item[(1)]平面$z=a$ $(0 \leqq a \leqq 1)$による$V$の切り口はどのような図形か．
\item[(2)]$V$の体積を求めよ．
\end{description}
