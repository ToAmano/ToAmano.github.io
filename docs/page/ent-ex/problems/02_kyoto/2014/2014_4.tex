　実数の定数$a$，$b$に対して，関数$f(x)$を
\[ f(x)=\frac{ax+b}{x^2+x+1} \]
で定める．すべての実数$x$で不等式
\[ f(x)\leqq{f(x)}^3-2{f(x)}^2+2 \]
が成り立つような点$(a, \, b)$の範囲を図示せよ．
