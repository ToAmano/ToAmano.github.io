　自然数$a$，$b$はどちらも3で割り切れないが，$a^3+b^3$は81で割り切れる．
このような$a$，$b$の組$(a, \, b)$のうち，$a^2+b^2$の値を最小にするものと，
そのときの$a^2+b^2$の値を求めよ．
