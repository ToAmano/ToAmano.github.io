{\huge 6}
　双曲線$\displaystyle y=\frac{1}{x}$の第1象限にある部分と，
原点$O$を中心とする円の第1象限にある部分を，
それぞれ$C_1$，$C_2$とする．
$C_1$と$C_2$は2つの異なる点$A$，$B$で交わり，
点$A$における$C_1$の接線$l$と線分$OA$のなす角は$\displaystyle\frac{\pi}{6}$であるとする．
このとき，$C_1$と$C_2$で囲まれる図形の面積を求めよ．
