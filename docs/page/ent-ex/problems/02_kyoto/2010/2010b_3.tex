{\huge 3}
　$a$を正の実数とする．
座標平面において曲線$y=\sin x$ $(0 \leqq x \leqq \pi)$と$x$軸とで囲まれた図形の面積を$S$とし，
曲線$y=\sin x$ $\displaystyle \left( 0 \leqq x \leqq \frac{\pi}{2} \right)$，
曲線$y=a\cos x$ $\displaystyle \left( 0 \leqq x \leqq \frac{\pi}{2} \right)$
および$x$軸で囲まれた図形の面積を$T$とする．
このとき$S:T=3:1$となるような$a$の値を求めよ．
\end{flushleft}
