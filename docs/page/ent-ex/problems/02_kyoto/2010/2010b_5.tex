　次の問に答えよ．
\begin{description}
\item[(1)]$n$を正の整数，$a=2^n$とする．
$3^a-1$は$2^{n+2}$で割り切れるが$2^{n+3}$では割り切れないことを示せ．
\item[(2)]$m$を正の偶数とする．
$3^m-1$が$2^m$で割り切れるならば$m=2$または$m=4$であることを示せ．
\end{description}
