　$a$，$b$を整数，$u$，$v$を有理数とする．
$u+v\sqrt3$が$x^2+ax+b=0$の解であるならば，$u$と$v$は共に整数であることを示せ．
ただし$\sqrt3$が無理数であることは使ってよい．
