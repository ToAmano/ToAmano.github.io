{\huge 3}
　座標平面上で，$x$座標と$y$座標が共に整数である点を格子点という．
$n$は自然数であるとして，不等式
\[ x>0, \hspace{1zw} y>0, \hspace{1zw} \log_2\frac{y}{x} \leqq x \leqq n \]
を満たす格子点の個数を求めよ．
