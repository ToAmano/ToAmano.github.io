{\huge 3}
　0以上の整数$x$に対して，$C(x)$で$x$の下2桁を表すことにする．
たとえば，$C(12578)=78$，$C(6)=6$である．
$n$を2でも5でも割り切れない正の整数とする．
\begin{description}
\item[(1)]$x$，$y$が0以上の整数のとき，$C(nx)=C(ny)$ならば，$C(x)=C(y)$であることを示せ．
\item[(2)]$C(nx)=1$となる0以上の整数$x$が存在することを示せ．
\end{description}
\end{flushleft}
