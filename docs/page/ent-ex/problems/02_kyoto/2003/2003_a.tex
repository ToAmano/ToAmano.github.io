　正三角形$ABC$の辺$AB$上に点$P_1$，$P_2$が，辺$BC$上に点$Q_1$，$Q_2$が，辺$CA$上に点$R_1$，$R_2$があり，
どの点も頂点には一致していないとする．
このとき三角形$P_1Q_1R_1$の重心と三角形$P_2Q_2R_2$の重心が一致すれば，
$P_1P_2=Q_1Q_2=R_1R_2$が成り立つことを示せ．
\end{flushleft}
