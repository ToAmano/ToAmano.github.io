\item[(1)]$n$を2以上の自然数とする．
複素数$z$が$z\neq1$，$z^n=1$をみたすとき，$1+2z+3z^2+\cdots+nz^{n-1}$は次の(ア)から(キ)のどれと等しくなるか．
根拠を示して1つ選べ．
\[
(ア) 0 \hspace{1zw}
(イ) n(z+1) \hspace{1zw}
(ウ) n(z-1) \hspace{1zw}
(エ) \frac{n}{z-1} \hspace{1zw}
(オ) \frac{n}{{(z-1)}^2} \hspace{1zw}
(カ) -\frac{2n}{{(z-1)}^2} \hspace{1zw}
(キ) 1-z-n \hspace{1zw}
\]
\item[(2)]次の等式が成り立つことを示せ．
\[ 2\sin{40}^\circ+3\sin{80}^\circ+\cdots+9\sin{320}^\circ=-\frac{9}{2\tan{20}^\circ} \]
