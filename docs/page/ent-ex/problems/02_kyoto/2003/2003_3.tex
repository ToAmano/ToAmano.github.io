　四面体$OABC$は次の2つの条件
\begin{description}
\item[(i)]$OA \bot BC$，$OB \bot AC$，$OC \bot AB$
\item[(ii)]4つの面の面積がすべて等しい
\end{description}
をみたしている．このとき，この四面体は正四面体であることを示せ．
