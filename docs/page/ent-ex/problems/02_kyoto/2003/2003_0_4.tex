{\huge 4}
　$p$は3以上の素数であり，$x$，$y$は$0 \leqq x \leqq p$，$0 \leqq y \leqq p$をみたす整数であるとする．
このとき$x^2$を$2p$で割った余りと，$y^2$を$2p$で割った余りが等しければ，$x=y$であることを示せ．
\end{flushleft}
