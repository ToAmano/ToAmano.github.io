{\huge 3}
　$xy$平面上の点で$x$座標，$y$座標がともに整数である点を格子点という．
\begin{description}
\item[(1)]格子点を頂点とする三角形の面積は$\displaystyle\frac{1}{2}$以上であることを示せ．
\item[(2)]格子点を頂点とする凸四角形の面積が1であるとき，この四角形は平行四辺形であることを示せ．
\end{description}
\end{flushleft}
