{\huge 3}
　$xy$平面上の点で$x$座標、$y$座標がともに整数である点を格子点という．
$a$，$k$は整数で$a\geqq2$とし，直線$L:ax+(a^2+1)y=k$を考える．
\begin{description}
\item[(1)]直線$L$上の格子点を1つ求めよ．
\item[(2)]$k=a(a^2+1)$のとき，$x>0$，$y>0$の領域に直線$L$上の格子点は存在しないことを示せ．
\item[(3)]$k>a(a^2+1)$ならば，$x>0$，$y>0$の領域に直線$L$上の格子点が存在することを示せ．
\end{description}
\end{flushleft}
