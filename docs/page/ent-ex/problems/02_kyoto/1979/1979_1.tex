{\huge 1}
　次の条件をみたしていて，かつ最高次の係数が1である$x$の整式$P_1(x)$，$P_2(x)$，$P_3(x)$を求めよ．
\begin{description}
\item[(i)]$P_1(x)$は1次式であって，
どんな定数$C$に対しても$\displaystyle\int_{-1}^1P_1(x)Cdx=0$
\item[(ii)]$P_2(x)$は2次式であって，
1次以下のどんな整式$f(x)$に対しても$\displaystyle\int_{-1}^1P_2(x)f(x)dx=0$
\item[(iii)]$P_3(x)$は3次式であって，
2次以下のどんな整式$f(x)$に対しても$\displaystyle\int_{-1}^1P_3(x)f(x)dx=0$
\end{description}
\end{flushleft}
