{\huge 5}
　正の定数$a$，$b$に対し，不等式$\displaystyle 4m<n^2<4m+\frac{a}{\sqrt{m}}+\frac{b}{m}$を考え，
次の問いに答えよ．
\begin{description}
\item[(イ)]$m>0$，かつ$m$，$n$ともに整数であって，
この不等式をみたすような$m$，$n$の組は有限個しか存在しないことを証明せよ．
\item[(ロ)]$a=8$，$b=9$，$m\geqq9$であるときは，
上の不等式をみたす整数$m$，$n$の組は$n^2=4m+1$をみたすことを証明せよ．
\item[(ハ)](ロ)の場合の$m$，$n$の組のうち，$n$が最も大きいものを求めよ．
\end{description}
\end{flushleft}
