# Condaコマンド
<!-- [備忘] python / conda環境構築の基本コマンド一覧 - Qiita -->

## 

- condaのアップデート
  ```bash
  conda update conda
  ```
- 仮想環境の作成
  ```bash
  conda create -n env_name
  ```
- 仮想環境の一覧表示
  ```bash
  conda info -e
  ```
- 仮想環境の切り替え
  ```python
  conda activate env_name
  ```
- 仮想環境から出る
  ```python
  conda deactivate 
  ```
